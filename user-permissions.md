# สิทธิ์การใช้งานของผู้ใช้

ระบบการจัดการสิ่งอำนวยความสะดวกปฏิบัติตามระบบควบคุมการเข้าถึงตามบทบาทของ iviva ที่เป็นมาตรฐาน

แอปพลิเคชันนี้กำหนดบทบาทของแอปพลิเคชันหลายบทบาท

โดยใช้กรอบการอนุญาตมาตรฐานของ iviva บทบาทเหล่านี้สามารถรวบรวมเข้าไปในบทบาทผู้ใช้และกลุ่มผู้ใช้



บทบาทของแอปพลิเคชันที่มีให้ใช้งานดังต่อไปนี้:

| บทบาทแอปพลิเคชัน                            | คำอธิบาย                                                                                                                                                                                                                                                                |
| ----------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ได้รับอนุญาตให้เข้าถึงแอปพลิเคชัน               | บทบาทนี้จำเป็นสำหรับทุกคนที่จะเข้าถึงแอปพลิเคชัน ผู้ใช้ที่มีบทบาทนี้สามารถดูข้อมูลวิเคราะห์ทั้งหมดในแอปพลิเคชัน                                                                                                                                           |
| ได้รับอนุญาตให้กำหนดค่าข้อมูล           | บทบาทนี้จำเป็นเพื่อให้ผู้ใช้สามารถกำหนดค่าข้อมูล - ซึ่งรวมถึงการกำหนดค่ามิเตอร์, มิเตอร์เสมือน, กลุ่มมิเตอร์, แท็ก, ฐานเก็บข้อมูลและสถานที่                                                                                                              |
| ได้รับอนุญาตให้บันทึกฟิลเตอร์และรายงาน | บทบาทนี้ให้สิทธิ์ให้ผู้ใช้สร้างรายงานในระบบและสร้างวิดเจ็ตแดชบอร์ดใหม่โดยใช้[consumption-analysis-tool.md](operational-guide/consumption-analysis-tool.md "mention") ที่ถูกตั้งค่าอย่างถาวรในแดชบอร์ดสำหรับทุกคนดู |
