# บทนำ

## ระบบการจัดการสาธารณูปโภค

ระบบการจัดการสาธารณูปโภคของ iviva (UMS) เป็นแอปพลิเคชันและบริการที่จัดการข้อมูลและให้การวิเคราะห์ที่เกี่ยวข้องกับทรัพยากรสาธารณูปโภค

ทรัพยากรสาธารณูปโภคคืออะไร?

โดยกว้าง ทรัพยากรที่สามารถ _บริโภค_ และ _วัดได้_ สามารถถือว่าเป็นทรัพยากรสาธารณูปโภค

ตัวอย่างบางอย่างคือ:

* ไฟฟ้า
* น้ำ
* ก๊าซ
* ขยะ
* แบนด์วิดธ์อินเทอร์เน็ต
* เชื้อเพลิง
* ออกซิเจน

นี้เป็นบางส่วนของทรัพยากรที่ข้อมูลสามารถจัดการได้ในแอปพลิเคชัน UMS.
