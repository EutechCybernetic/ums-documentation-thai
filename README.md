# บทนำ

## ระบบการจัดการสาธารณูปโภค

ระบบการจัดการสาธารณูปโภค iviva (UMS) เป็นแอปพลิเคชันและบริการในการจัดการข้อมูลและให้ข้อมูลวิเคราะห์ที่เกี่ยวข้องกับทรัพยากรสาธารณูปโภค

ทรัพยากรสาธารณูปโภคคืออะไร?

โดยกว้าง ๆ แล้ว - ทรัพยากรใด ๆ ที่สามารถ _บริโภค_ และ _วัดได้_ อาจถือว่าเป็นทรัพยากรสาธารณูปโภค

ตัวอย่างบางอย่างมีดังนี้:

* ไฟฟ้า
* น้ำ
* ก๊าซ
* ขยะ
* แบนด์วิดธ์อินเทอร์เน็ต
* เชื้อเพลิง
* ออกซิเจน

นี่คือบางส่วนของทรัพยากรที่ข้อมูลสามารถจัดการได้ในแอปพลิเคชัน UMS.
