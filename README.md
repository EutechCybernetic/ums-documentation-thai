# บทนำ

## ระบบการจัดการสาธารณูปโภค

ระบบการจัดการสาธารณูปโภค (UMS) ของ iviva เป็นแอปพลิเคชันและบริการที่จัดการข้อมูลและให้วิเคราะห์ที่เกี่ยวข้องกับทรัพยากรสาธารณูปโภค

ทรัพยากรสาธารณูปโภคคืออะไร?

โดยกว้าง ๆ แล้ว - ทรัพยากรใด ๆ ที่สามารถ _บริโภค_ และ _วัดได้_ สามารถถือว่าเป็นทรัพยากรสาธารณูปโภค

ตัวอย่างบางส่วนได้แก่:

* ไฟฟ้า
* น้ำ
* ก๊าซ
* ขยะ
* แบนด์วิดธ์อินเทอร์เน็ต
* เชื้อเพลิง
* ออกซิเจน

เหล่านี้เป็นบางส่วนของทรัพยากรที่ข้อมูลสามารถจัดการได้ในแอปพลิเคชัน UMS.
