# การตั้งค่าประเภทยูทิลิตี้

คุณสามารถตั้งค่าประเภทยูทิลิตี้หลายประเภทในแอปพลิเคชัน

ในส่วน Configuration ไปที่แท็บ 'Meter Types' เพื่อตั้งค่าประเภทยูทิลิตี้ต่างๆ

{% hint style="info" %}
เราใช้คำว่า 'meter type' และ 'utility type' แลกเปลี่ยนกัน
{% endhint %}

ที่นี่คุณสามารถดูประเภทยูทิลิตี้ที่มีอยู่แล้ว แก้ไข และเพิ่มประเภทใหม่

คลิกที่ปุ่ม 'Add Item' ที่มุมขวาบนเพื่อเพิ่มประเภทยูทิลิตี้ใหม่

คุณสามารถคลิกที่ปุ่ม 'edit' ที่อยู่ใต้ประเภทยูทิลิตี้เพื่อแก้ไขคุณสมบัติของมัน

คุณยังสามารถลบประเภทยูทิลิตี้โดยการคลิกที่ไอคอนลบ (ถังขยะ)

{% hint style="warning" %}
การลบเหมือนกับเพชร คือตลอดกาล โปรดอย่าลบประเภทยูทิลิตี้ ยกเว้นคุณสร้างมันโดยไม่ได้ตั้งใจเมื่อสักครู่
{% endhint %}



## การตั้งค่าประเภทยูทิลิตี้

ไม่ว่าคุณจะเพิ่มประเภทยูทิลิตี้ใหม่หรือแก้ไขที่มีอยู่แล้ว หน้าตาของอินเทอร์เฟซในการทำงานนั้นคล้ายกัน

คุณมีตัวเลือกในการระบุหน่วยที่ได้รับการสนับสนุนสำหรับประเภทยูทิลิตี้นี้

การตั้งค่าหน่วยเป็นสิ่งสำคัญ คุณต้องการอย่างน้อยหนึ่งหน่วยสำหรับแต่ละประเภทยูทิลิตี้



## ความเข้าใจเกี่ยวกับหน่วย

แต่ละประเภทยูทิลิตี้สามารถสนับสนุนหน่วยหลายหน่วยที่บันทึกการบริโภค

สำหรับไฟฟ้า - นี้อาจเป็น `mWh` หรือ `kWh`

สำหรับน้ำเย็น นี้อาจเป็น `mWh` หรือ `btu`

สำหรับการบริโภคขยะ อาจเป็น \`kg\`

สำหรับน้ำ อาจเป็นลิตรหรือแกลลอน

แต่ละประเภทยูทิลิตี้มีหนึ่ง **หน่วยพื้นฐาน**

แล้วหน่วยอื่น ๆ ถูกกำหนดในความสัมพันธ์กับหน่วยพื้นฐาน

แต่ละหน่วยถูกกำหนดด้วย _factor การแปลง_ ที่กำหนดวิธีการแปลงเป็นหน่วยพื้นฐาน

หน่วยพื้นฐานมี factor การแปลงเป็น 1 เสมอ

{% hint style="info" %}
วิธีที่คุณกำหนดหน่วยพื้นฐานคือโดยการกำหนดหน่วยใหม่และให้ factor การแปลงเป็น 1 นั้นจะทำให้มันเป็นหน่วยพื้นฐานของคุณโดยอัตโนมัติ
{% endhint %}

{% hint style="info" %}
คุณสามารถมีหน่วยพื้นฐานหลายหน่วยได้หรือไม่ โดยการกำหนดหน่วยหลายหน่วยที่มี factor การแปลงเป็น 1 จริงๆ คุณสามารถทำได้ แต่จริงๆ แล้วมันไม่มีประโยชน์
{% endhint %}



หน่วยพื้นฐานเป็นสิ่งสำคัญ - โดยค่าเริ่มต้นข้อมูลทั้งหมดจะถูกแปลงเป็นหน่วยพื้นฐานเมื่อบันทึกและหน่วยเริ่มต้นสำหรับการรายงานจะเป็นหน่วยพื้นฐาน
