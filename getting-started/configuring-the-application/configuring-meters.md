# การตั้งค่ามิเตอร์

เมื่อคุณได้ตั้งค่าประเภทสาธารณูปโภคของคุณแล้ว คุณสามารถเริ่มเพิ่มและตั้งค่ามิเตอร์ได้

{% hint style="info" %}
ที่นี่ 'มิเตอร์' หมายถึงเมตรทางกายภาพ - ซึ่งจะบันทึกข้อมูลการบริโภค
{% endhint %}

ไปที่ส่วนการตั้งค่าในแอปพลิเคชัน และใต้ 'มิเตอร์' คุณสามารถเริ่มเพิ่มมิเตอร์ได้

ส่วนมิเตอร์แสดงรายการของมิเตอร์ทางกายภาพทั้งหมดที่คุณได้ตั้งค่าไว้ คุณสามารถคลิก 'เพิ่มรายการ' ที่ด้านบนเพื่อลงทะเบียนมิเตอร์ใหม่ คุณสามารถแก้ไขแต่ละมิเตอร์โดยคลิกที่ไอคอน 'แก้ไข' ที่ด้านขวาของแต่ละมิเตอร์

## การตั้งค่ามิเตอร์

เมื่อเพิ่มมิเตอร์ใหม่ คุณต้องระบุรายละเอียดต่อไปนี้:

1. ชื่อมิเตอร์ - ชื่อเพื่อช่วยคุณระบุมิเตอร์นี้ คุณสามารถให้ชื่อที่คุณต้องการ แต่ควรเป็นเอกลักษณ์ คุณจะไม่ได้รับอนุญาตให้สร้างมิเตอร์หลายคันที่มีชื่อเดียวกัน
2. คำอธิบาย - คำอธิบายสั้น ๆ ของมิเตอร์นี้ - เพื่อความสะดวกของคุณเท่านั้น แอปพลิเคชันนี้ไม่ได้ใช้
3. ประเภทมิเตอร์ - ประเภทมิเตอร์นี้คืออะไร - คุณจะเลือกหนึ่งในประเภทสาธารณูปโภคที่คุณได้ตั้งค่าไว้ก่อนหน้านี้ (ตัวอย่าง: พลังงาน หรือน้ำ หรือขยะ)
4. หน่วย - หน่วยที่มิเตอร์นี้บันทึกข้อมูล นี่เป็นสิ่งที่สำคัญเพราะเมื่อข้อมูลถูกอัปโหลด/ส่ง/บันทึกสำหรับมิเตอร์นี้ ข้อมูลจะถูกถือว่าอยู่ในหน่วยที่คุณระบุที่นี่ ดังนั้นหากหน่วยพื้นฐานของคุณคือ 'กิโลวัตต์-ชั่วโมง' แต่มิเตอร์ที่กำลังถูกตั้งค่าใช้ btu ให้ระบุ btu เป็นหน่วย  เมื่อข้อมูลถูกบันทึก ข้อมูลจะถูกแปลงจาก btu เป็นกิโลวัตต์-ชั่วโมงโดยอัตโนมัติเมื่อข้อมูลถูกรับและเก็บรักษา
5. สถานที่ให้บริการ - สถานที่ที่มิเตอร์นี้ให้บริการ เมื่อคุณวิเคราะห์ข้อมูลการบริโภคตามสถานที่ ข้อมูลสถานที่นี้จะถูกใช้เพื่อกำหนดว่ามิเตอร์ใดบริการและสนับสนุนสถานที่เฉพาะ
6. แท็ก - คุณสามารถใช้แท็กหลายตัวกับมิเตอร์ แท็กต้องถูกกำหนดไว้ล่วงหน้า ดูส่วน [configuring-tags.md](configuring-tags.md "mention") สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการทำงานของแท็ก\
   เพื่อเพิ่มแท็กใหม่ คลิกที่ปุ่ม 'เพิ่มแท็ก' จากนั้นเลือกประเภทแท็ก\
   เมื่อคุณเลือกประเภทแท็ก - คุณจะได้รับรายการของแท็กที่เป็นไปได้สำหรับประเภทแท็กนั้น คุณสามารถเลือกหนึ่งเพื่อใช้\
   คุณสามารถเลือกแท็กหนึ่งของแต่ละประเภท แต่คุณสามารถเพิ่มประเภทแท็กที่คุณต้องการได้ไม่จำกัด
7. ข้อมูลพื้นฐาน คุณสามารถกำหนดค่าพื้นฐานสำหรับแต่ละพื้นฐานที่คุณได้กำหนดไว้ พวกเขาเป็นค่าพื้นฐานการบริโภคประจำวัน ดูส่วนการตั้งค่าพื้นฐานสำหรับข้อมูลเพิ่มเติม
   1. Metadata - คุณสามารถกำหนดค่าฟิลด์ที่กำหนดเองต่าง ๆ ภายใต้ส่วน 'metadata' ฟิลด์ที่กำหนดเองถูกกำหนดไว้ภายใต้ส่วนการตั้งค่า metadata ซึ่งอยู่ในส่วน 'ขั้นสูง' &#x20;

{% hint style="info" %}
การเปลี่ยนแปลงทั้งหมดจะถูกบันทึกเฉพาะเมื่อคุณกดปุ่ม 'ส่ง' การแก้ไขของคุณจะไม่ถูกบันทึกจนกว่าคุณจะทำเช่นนั้น
{% endhint %}
