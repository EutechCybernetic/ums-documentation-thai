# การตั้งค่ามิเตอร์เสมือน

การตั้งค่ามิเตอร์เสมือนเหมือนกับการตั้งค่ามิเตอร์ทางกายภาพ ยกเว้นว่ามีการตั้งค่าเพิ่มเติมสำหรับสูตรของมิเตอร์เสมือน

สูตรถูกกำหนดโดยการเลือกมิเตอร์ทางกายภาพหนึ่งหรือมากกว่าและปรับสเกลให้กับแต่ละมิเตอร์ (สเกลสามารถเป็นลบหรือเป็นส่วนหนึ่ง)

ดังนั้นถ้าคุณต้องการกำหนดมิเตอร์เสมือนให้มีผลรวมจากมิเตอร์ A และ B และลบด้วยมิเตอร์ C, คุณจะต้องเพิ่มมิเตอร์และสเกลดังต่อไปนี้:

| มิเตอร์ | สเกล |
| ----- | -------------- |
| A     | 1              |
| B     | 1              |
| C     | -1             |

สิ่งนี้จะทำให้สูตร `Ax1 + Bx1  + Cx-1` ถูกใช้สำหรับมิเตอร์เสมือนนั้น
