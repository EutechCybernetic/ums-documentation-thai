# การกำหนดค่ามิเตอร์เสมือน

การกำหนดค่ามิเตอร์เสมือนเหมือนกับการกำหนดค่ามิเตอร์ทางกายภาพ ยกเว้นว่ามีการกำหนดค่าเพิ่มเติมสำหรับสูตรของมิเตอร์เสมือน



สูตรถูกกำหนดโดยการเลือกมิเตอร์ทางกายภาพหนึ่งหรือมากกว่าและปรับสเกลแต่ละมิเตอร์ (ปัจจัยการปรับสเกลสามารถเป็นค่าลบหรือเศษส่วน)

ดังนั้นถ้าคุณต้องการกำหนดมิเตอร์เสมือนเป็นผลรวมจากมิเตอร์ A และ B และลบด้วยมิเตอร์ C, คุณจะต้องเพิ่มมิเตอร์และปัจจัยการปรับสเกลต่อไปนี้:

| มิเตอร์ | ปัจจัยการปรับสเกล |
| ----- | -------------- |
| A     | 1              |
| B     | 1              |
| C     | -1             |

สิ่งนี้จะทำให้สูตร `Ax1 + Bx1  + Cx-1` ถูกนำไปใช้สำหรับมิเตอร์เสมือนนั้น
