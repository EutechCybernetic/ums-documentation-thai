# การกำหนดค่ามิเตอร์เสมือน

การกำหนดค่ามิเตอร์เสมือนนั้นเหมือนกับการกำหนดค่ามิเตอร์ทางกายภาพ ยกเว้นว่ามีการกำหนดค่าเพิ่มเติมสำหรับสูตรของมิเตอร์เสมือน



สูตรถูกกำหนดโดยการเลือกมิเตอร์ทางกายภาพหนึ่งหรือมากกว่าและนำมาคูณด้วยสเกลลิ่งแฟคเตอร์ (สเกลลิ่งแฟคเตอร์สามารถเป็นค่าลบหรือเศษส่วน).

ดังนั้น ถ้าคุณต้องการกำหนดมิเตอร์เสมือนเป็นผลรวมของมิเตอร์ A และ B และลบด้วยมิเตอร์ C, คุณจะต้องเพิ่มมิเตอร์และสเกลลิ่งแฟคเตอร์ตามนี้:

| มิเตอร์ | สเกลลิ่งแฟคเตอร์ |
| ----- | -------------- |
| A     | 1              |
| B     | 1              |
| C     | -1             |

สิ่งนี้จะทำให้ได้สูตร `Ax1 + Bx1  + Cx-1` ที่ถูกนำไปใช้สำหรับมิเตอร์เสมือนนั้น
