# การกำหนดค่ามิเตอร์เสมือน

การกำหนดค่ามิเตอร์เสมือนเหมือนกับการกำหนดค่ามิเตอร์ทางกายภาพ ยกเว้นว่ามีการกำหนดค่าเพิ่มเติมสำหรับสูตรของมิเตอร์เสมือน

สูตรถูกกำหนดโดยการเลือกมิเตอร์ทางกายภาพหนึ่งหรือมากกว่าและใช้สเกลแฟคเตอร์กับแต่ละมิเตอร์ (สเกลแฟคเตอร์สามารถเป็นลบหรือเป็นส่วนหนึ่ง)

ดังนั้นถ้าคุณต้องการกำหนดมิเตอร์เสมือนเพื่อให้ได้ผลรวมจากมิเตอร์ A และ B และลบด้วยมิเตอร์ C, คุณจะต้องเพิ่มมิเตอร์และสเกลแฟคเตอร์ดังต่อไปนี้:

| มิเตอร์ | สเกลแฟคเตอร์ |
| ----- | -------------- |
| A     | 1              |
| B     | 1              |
| C     | -1             |

สิ่งนี้จะทำให้ได้สูตร `Ax1 + Bx1  + Cx-1` ที่ใช้สำหรับมิเตอร์เสมือนนั้น
