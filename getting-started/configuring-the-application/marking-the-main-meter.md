# การทำเครื่องหมายเมตรหลัก

เมื่อคุณมีเมตรแม่และเมตรย่อย คุณอาจจำเป็นต้องทำเครื่องหมายเมตรที่เฉพาะเจาะจงเป็นเมตรหลักสำหรับสถานที่นั้น ซึ่งจะใช้เพื่อติดตามการบริโภคในสถานที่นั้น

ในการทำสิ่งนี้ ไปที่ 'Advanced' ในแถบด้านข้าง แล้วไปที่แท็บ 'Meter To Tag Mapping'

ที่นี่ เลือกประเภทสาธารณูปโภค แล้วเพิ่มการทำแผนที่ใหม่

เลือก 'Location' เป็นประเภทแท็ก แล้วเลือกสถานที่

ภายใต้ 'Resource Type' เลือกประเภทของเมตร (เมตรปกติ, เมตรเสมือนจริง หรือกลุ่มเมตร) ที่คุณต้องการทำแผนที่ไปยังสถานที่

จากนั้น ในส่วน 'Resource' เลือกเมตรหรือกลุ่มที่เฉพาะเจาะจงที่คุณต้องการทำแผนที่

ส่งการเปลี่ยนแปลงของคุณ

เมตรนี้ (หรือกลุ่ม) ตอนนี้เป็นเมตรหลักสำหรับสถานที่นั้น

เมื่อคุณเลือกสถานที่สำหรับดูข้อมูลการบริโภค เมตรนี้จะถูกใช้เพื่อดูการบริโภค



{% hint style="info" %}
สิ่งนี้ใช้เฉพาะกับสถานที่นั้น ไม่ใช่สถานที่ย่อยใดๆ คุณต้องทำแผนที่เมตรหลักแยกต่างหากสำหรับสถานที่ย่อย ถ้าสถานที่ย่อยนั้นยังมีบอร์ดจ่ายไฟหรือประเภทอื่นๆ ของเมตรแม่ในตัว
