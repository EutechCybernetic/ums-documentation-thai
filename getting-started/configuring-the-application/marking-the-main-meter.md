# การทำเครื่องหมายเมตรหลัก

เมื่อคุณมีเมตรหลักและเมตรย่อย คุณอาจจำเป็นต้องทำเครื่องหมายเมตรที่เฉพาะเจาะจงเป็นเมตรหลักสำหรับสถานที่นั้นซึ่งจะใช้ในการติดตามการบริโภคในสถานที่นั้น

เพื่อทำสิ่งนี้ ไปที่ 'ขั้นสูง' ในแถบด้านข้าง แล้วไปที่แท็บ 'การจับคู่เมตรกับแท็ก'

ที่นี่ เลือกประเภทสาธารณูปโภค แล้วเพิ่มการจับคู่ใหม่

เลือก 'สถานที่' เป็นประเภทแท็ก แล้วเลือกสถานที่

ภายใต้ 'ประเภททรัพยากร' เลือกประเภทเมตร (เมตรปกติ เมตรเสมือนจริง หรือกลุ่มเมตร) ที่คุณต้องการจับคู่กับสถานที่

จากนั้น ในส่วน 'ทรัพยากร' เลือกเมตรหรือกลุ่มที่เฉพาะเจาะจงที่คุณต้องการจับคู่

ส่งการเปลี่ยนแปลงของคุณ

เมตรนี้ (หรือกลุ่ม) ตอนนี้เป็นเมตรหลักสำหรับสถานที่นั้น

เมื่อคุณเลือกสถานที่สำหรับดูข้อมูลการบริโภค เมตรนี้จะถูกใช้ในการบริโภค



{% hint style="info" %}
สิ่งนี้ใช้ได้เฉพาะกับสถานที่นั้น ไม่ใช่สถานที่ย่อยใดๆ คุณต้องจับคู่เมตรหลักแยกต่างหากสำหรับสถานที่ย่อย หากสถานที่ย่อยเหล่านั้นยังมีบอร์ดจ่ายไฟหรือประเภทเมตรหลักอื่นๆ อยู่ในนั้น
