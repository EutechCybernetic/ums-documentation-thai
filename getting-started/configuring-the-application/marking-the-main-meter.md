# การทำเครื่องหมายเมตรหลัก

เมื่อคุณมีเมตรแม่และเมตรย่อย คุณอาจจำเป็นต้องทำเครื่องหมายเมตรเฉพาะเป็นเมตรหลักสำหรับสถานที่นั้นซึ่งจะใช้ในการติดตามการบริโภคในสถานที่นั้น

ในการทำสิ่งนี้ ไปที่ 'ขั้นสูง' ในแถบข้าง แล้วไปที่แท็บ 'การจับคู่เมตรกับแท็ก'

ที่นี่ เลือกประเภทยูทิลิตี้ แล้วเพิ่มการจับคู่ใหม่

เลือก 'สถานที่' เป็นประเภทแท็ก แล้วเลือกสถานที่

ใต้ 'ประเภททรัพยากร' เลือกประเภทเมตร (เมตรปกติ, เมตรเสมือนจริง หรือกลุ่มเมตร) ที่คุณต้องการจับคู่กับสถานที่

จากนั้น ในส่วน 'ทรัพยากร' เลือกเมตรเฉพาะหรือกลุ่มที่คุณต้องการจับคู่

ส่งการเปลี่ยนแปลงของคุณ

เมตรนี้ (หรือกลุ่ม) ตอนนี้เป็นเมตรหลักสำหรับสถานที่นั้น

เมื่อคุณเลือกสถานที่สำหรับดูข้อมูลการบริโภค เมตรนี้จะถูกใช้ในการบริโภค



{% hint style="info" %}
สิ่งนี้ใช้ได้เฉพาะกับสถานที่นั้น ไม่ใช่สถานที่ย่อยใดๆ คุณต้องจับคู่เมตรหลักแยกกับสถานที่ย่อย ถ้าสถานที่ย่อยเหล่านั้นยังมีบอร์ดการกระจายหรือประเภทอื่นๆ ของเมตรแม่อยู่ในนั้น
