# การเข้าใจหน้าผู้ใช้งาน

_หมายเหตุ: เช่นเดียวกับทุกๆ แอปพลิเคชัน, UMS จะถูกอัปเดตและปรับปรุงอย่างต่อเนื่องและหน้าตาของหน้าผู้ใช้งานอาจจะมีการเปลี่ยนแปลงเล็กน้อยระหว่างเวอร์ชัน - แต่แนวคิดทั้งหมดยังคงเหมือนเดิม_

หน้าตาหลักของแอปพลิเคชัน UMS จะมีลักษณะดังนี้:

<figure><img src="../../.gitbook/assets/image (11).png" alt=""><figcaption></figcaption></figure>

ด้านขวาคือพื้นที่เนื้อหาและด้านซ้ายคือแถบนำทางของคุณ

คุณสามารถใช้แถบนำทางเพื่อเลือกเมนูต่างๆ ของแอปพลิเคชันและเนื้อหาสำหรับแต่ละส่วนจะอยู่ทางด้านขวา



ส่วนหลักของแอปพลิเคชันมีดังนี้:

1. แดชบอร์ด - สามารถตั้งค่าแดชบอร์ดหลายๆ แดชบอร์ดและแต่ละแดชบอร์ดสามารถมีวิดเจ็ตที่สามารถปรับแต่งได้
2. การกำหนดค่า - ส่วนนี้ของแอปพลิเคชันจะช่วยให้คุณสามารถกำหนดค่าองค์ประกอบต่างๆ ของแอปพลิเคชัน - การกำหนดค่าเมตรและแท็กและการกำหนดค่าที่เกี่ยวข้องทั้งหมดจะทำที่นี่
3. ขั้นสูง - การเข้าถึงคุณสมบัติการดำเนินการและระดับผู้ดูแลระบบขั้นสูง
4. ประเภทยูทิลิตี้ - แต่ละประเภทยูทิลิตี้จะมีส่วนของตัวเองในแถบนำทางที่ช่วยให้คุณสามารถตรวจสอบ
   1. เมตรที่เกี่ยวข้องกับประเภทยูทิลิตี้นั้น
   2. สำรวจแท็กที่เกี่ยวข้องกับประเภทยูทิลิตี้นั้น
