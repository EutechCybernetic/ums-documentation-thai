# แท็ก กับ กลุ่มมิเตอร์

ทั้งแท็กและกลุ่มมิเตอร์ทำให้คุณสามารถรวมข้อมูลจากหลายมิเตอร์ได้ - ดังนั้นความแตกต่างคืออะไร และเราควรใช้แท็กเมื่อไหร่ และใช้กลุ่มมิเตอร์เมื่อไหร่?

มีหลายที่ที่คุณสามารถเลือกใช้ได้

ในตัวอย่างด้านบน คุณสามารถสร้างกลุ่มมิเตอร์สำหรับ 'Sunbucks ชั้นล่าง' และ 'Sunbucks ชั้น 3' แล้วใช้แทน แล้วสร้างอีกหนึ่งสำหรับ 'Sunbucks' ที่เพิ่มชั้นล่างและชั้น 3

ข้อดีของแท็กคือการตั้งค่าและกำหนดค่าที่ง่ายกว่า นอกจากนี้ยังง่ายต่อการดูและวิเคราะห์ข้อมูลในโครงสร้างแบบต้นไม้ มันให้คุณมีความสัมพันธ์ที่ชัดเจน ในขณะที่กลุ่มมิเตอร์เป็นรายการแบบแบนของวัตถุ

อย่างไรก็ตาม หากตรรกะการจัดกลุ่มของคุณไม่ได้เพียงแค่การรวมกัน - หากคุณต้องใช้ปัจจัยการปรับขนาดหรือลบค่าการบริโภคมิเตอร์บางอย่าง คุณไม่สามารถใช้แท็กเพียงอย่างเดียวได้ คุณต้องใช้มิเตอร์เสมือนหรือกลุ่มมิเตอร์

(คุณสามารถแท็กกลุ่มมิเตอร์นั้นในภายหลัง)
