# แท็ก ประกอบกับ กลุ่มมิเตอร์



ทั้งแท็กและกลุ่มมิเตอร์ช่วยให้คุณสามารถรวบรวมข้อมูลจากมิเตอร์หลายๆ เครื่อง - ดังนั้นความแตกต่างคืออะไรและเราควรใช้แท็กเมื่อไหร่และใช้กลุ่มมิเตอร์เมื่อไหร่?

มีหลายที่ที่คุณสามารถเลือกที่จะใช้ทั้งสองอย่าง

ในตัวอย่างด้านบนคุณสามารถสร้างกลุ่มมิเตอร์สำหรับ 'Sunbucks ชั้นล่าง' และ 'Sunbucks ชั้น 3' แล้วใช้แทน แล้วสร้างอีกหนึ่งสำหรับ 'Sunbucks' ที่รวมชั้นล่างและชั้น 3

ข้อดีของแท็กคือมันง่ายต่อการตั้งค่าและกำหนดค่า มันยังง่ายต่อการดูและวิเคราะห์ข้อมูลในโครงสร้างแบบต้นไม้ มันให้ความสัมพันธ์ที่ชัดเจนในขณะที่กลุ่มมิเตอร์เป็นรายการแบบแบนของวัตถุ

อย่างไรก็ตาม หากตรรกะการจัดกลุ่มของคุณไม่ได้เพียงแค่การรวมเข้าด้วยกัน - หากคุณต้องการใช้ปัจจัยการปรับขนาดหรือลบค่าการใช้มิเตอร์บางอย่าง แล้วคุณไม่สามารถใช้แท็กเพียงอย่างเดียวได้ คุณต้องใช้มิเตอร์เสมือนหรือกลุ่มมิเตอร์

(คุณสามารถแท็กกลุ่มมิเตอร์นั้นต่อไป)
