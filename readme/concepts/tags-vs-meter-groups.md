# แท็ก ประกัน กลุ่มมิเตอร์

ทั้งแท็กและกลุ่มมิเตอร์ทั้งสองช่วยให้คุณสามารถรวบรวมข้อมูลจากหลายๆ มิเตอร์ - ดังนั้นความแตกต่างคืออะไรและเราควรใช้แท็กหรือกลุ่มมิเตอร์เมื่อไหร่?

มีสถานที่หลายที่ที่คุณสามารถเลือกที่จะใช้ทั้งสองอย่าง

ในตัวอย่างด้านบนคุณสามารถสร้างกลุ่มมิเตอร์สำหรับ 'Sunbucks ชั้นล่าง' และ 'Sunbucks ชั้น 3' และใช้แทน แล้วสร้างอีกหนึ่งสำหรับ 'Sunbucks' ที่รวม Ground Floor และ 3rd Floor

ข้อดีของแท็กคือมันง่ายต่อการตั้งค่าและกำหนดค่า นอกจากนี้ยังง่ายต่อการดูและวิเคราะห์ข้อมูลในโครงสร้างแบบต้นไม้ มันให้คุณมีความสัมพันธ์ที่ชัดเจนในขณะที่กลุ่มมิเตอร์เป็นรายการแบบแบนของวัตถุ

อย่างไรก็ตาม หากตรรกะการจัดกลุ่มของคุณไม่ได้เพียงแค่การรวมกัน - หากคุณต้องใช้ปัจจัยการปรับขนาดหรือลบค่าการบริโภคมิเตอร์บางอย่าง คุณไม่สามารถใช้แท็กเพียงอย่างเดียวสำหรับสิ่งนั้น คุณต้องใช้มิเตอร์เสมือนหรือกลุ่มมิเตอร์

(คุณสามารถแท็กกลุ่มมิเตอร์นั้นต่อไป)
