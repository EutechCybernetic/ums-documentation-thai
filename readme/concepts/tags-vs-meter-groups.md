# แท็ก ประกอบกับ กลุ่มมิเตอร์

ทั้งแท็กและกลุ่มมิเตอร์ช่วยให้คุณสามารถรวบรวมข้อมูลจากหลายมิเตอร์ - ดังนั้นความแตกต่างคืออะไรและเราควรใช้แท็กหรือกลุ่มมิเตอร์เมื่อไหร่?

มีสถานที่มากมายที่คุณสามารถเลือกใช้ได้

ในตัวอย่างด้านบนคุณสามารถสร้างกลุ่มมิเตอร์สำหรับ 'Sunbucks Ground Floor' และ 'Sunbucks 3rd Floor' และใช้แทน แล้วสร้างอีกหนึ่งสำหรับ 'Sunbucks' ที่เพิ่ม Ground Floor และ 3rd Floor

ข้อดีของแท็กคือง่ายต่อการตั้งค่าและกำหนดค่า นอกจากนี้ยังง่ายต่อการดูและวิเคราะห์ข้อมูลในโครงสร้างแบบต้นไม้ มันให้คุณมีความสัมพันธ์ที่ชัดเจนในขณะที่กลุ่มมิเตอร์เป็นรายการแบบแบนของวัตถุ

อย่างไรก็ตาม หากตรรกะการจัดกลุ่มของคุณไม่ใช่การเพิ่มขึ้นเท่านั้น - ถ้าคุณต้องใช้ปัจจัยการปรับขนาดหรือลบค่าการบริโภคมิเตอร์บางอย่าง คุณไม่สามารถใช้แท็กอย่างเดียวสำหรับสิ่งนั้น คุณต้องใช้มิเตอร์เสมือนหรือกลุ่มมิเตอร์

(คุณสามารถแท็กกลุ่มมิเตอร์นั้นได้ในภายหลัง)
