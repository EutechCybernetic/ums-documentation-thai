# แท็ก ประกอบกับ กลุ่มมิเตอร์



ทั้งแท็กและกลุ่มมิเตอร์ทำให้คุณสามารถรวมข้อมูลจากหลายมิเตอร์ - ดังนั้นความแตกต่างคืออะไรและเราควรใช้แท็กเมื่อไหร่และใช้กลุ่มมิเตอร์เมื่อไหร่?

มีหลายที่ที่คุณสามารถเลือกใช้ระหว่างสองอย่างได้

ในตัวอย่างด้านบนคุณสามารถสร้างกลุ่มมิเตอร์สำหรับ 'Sunbucks ชั้นล่าง' และ 'Sunbucks ชั้น 3' แล้วใช้แทน แล้วสร้างอีกหนึ่งสำหรับ 'Sunbucks' ที่เพิ่มชั้นล่างและชั้น 3

ข้อดีของแท็กคือมันง่ายต่อการตั้งค่าและกำหนดค่า นอกจากนี้ยังง่ายต่อการดูและวิเคราะห์ข้อมูลในโครงสร้างแบบต้นไม้ มันให้คุณมีความสัมพันธ์ที่ชัดเจนในขณะที่กลุ่มมิเตอร์เป็นรายการแบบแบนของวัตถุ

อย่างไรก็ตาม หากตรรกะการจัดกลุ่มของคุณไม่ได้เพียงแค่การเพิ่มเข้าด้วยกัน - หากคุณต้องใช้ปัจจัยการปรับขนาดหรือลบค่าการบริโภคมิเตอร์บางอย่าง คุณไม่สามารถใช้แท็กเพียงอย่างเดียวได้ คุณต้องใช้มิเตอร์เสมือนจริงหรือกลุ่มมิเตอร์

(คุณสามารถแท็กกลุ่มมิเตอร์นั้นต่อไป)
