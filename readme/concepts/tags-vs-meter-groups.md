# แท็ก ประกอบกับ กลุ่มมิเตอร์

ทั้งแท็กและกลุ่มมิเตอร์ทำให้คุณสามารถรวบรวมข้อมูลจากหลายมิเตอร์ - ดังนั้นความแตกต่างคืออะไรและเราควรใช้แท็กหรือกลุ่มมิเตอร์เมื่อไหร่?

มีสถานที่มากมายที่คุณสามารถเลือกที่จะใช้ทั้งสองอย่าง

ในตัวอย่างด้านบนคุณสามารถสร้างกลุ่มมิเตอร์สำหรับ 'Sunbucks ชั้นล่าง' และ 'Sunbucks ชั้นที่ 3' และใช้แทน แล้วสร้างอีกหนึ่งสำหรับ 'Sunbucks' ที่เพิ่มชั้นล่างและชั้นที่ 3

ข้อดีของแท็กคือการตั้งค่าและกำหนดค่าที่ง่ายขึ้น มันยังง่ายต่อการดูและวิเคราะห์ข้อมูลในโครงสร้างแบบต้นไม้ มันให้ความสัมพันธ์ที่ชัดเจนในขณะที่กลุ่มมิเตอร์เป็นรายการแบบแบนของวัตถุ

อย่างไรก็ตาม หากตรรกะการจัดกลุ่มของคุณไม่ได้เพียงการรวมกัน - ถ้าคุณต้องใช้ปัจจัยการปรับขนาดหรือลบค่าการบริโภคมิเตอร์บางอย่าง แล้วคุณไม่สามารถใช้แท็กอย่างเดียวสำหรับสิ่งนั้น คุณต้องใช้มิเตอร์เสมือนหรือกลุ่มมิเตอร์

(คุณสามารถแท็กกลุ่มมิเตอร์นั้นในภายหลัง)
