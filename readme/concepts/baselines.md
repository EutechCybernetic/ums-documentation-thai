# ฐานศูนย์

ฐานศูนย์ช่วยให้คุณสามารถติดตามการบริโภคจริงของคุณเมื่อเทียบกับการบริโภคที่เป็นไอเดีย

การบริโภคที่เหมาะสมคืออะไร? มันขึ้นอยู่กับความต้องการของคุณ!

การบริโภคที่เหมาะสมของคุณอาจจะเป็นสิ่งที่คุณบันทึกเป็นการบริโภคในจุดเวลาที่เฉพาะเจาะจงในอดีตและคุณต้องการเปรียบเทียบว่าการบริโภคปัจจุบันของคุณติดตามตามนั้นอย่างไร

การบริโภคที่เหมาะสมของคุณอาจจะขึ้นอยู่กับมาตรฐานที่มีอยู่ตามกฎหมายหรือมาตรฐานการปฏิบัติที่คุณกำลังมองหาเพื่อปฏิบัติตาม

คุณสามารถกำหนดฐานศูนย์หลายๆ อัน - บางอันอาจจะขึ้นอยู่กับมาตรฐานภายในของคุณเกี่ยวกับการบริโภคที่ควรจะเป็นอย่างไรและอื่น ๆ อาจจะเพื่อติดตามตามมาตรฐานของประเทศหรือมาตรฐานระดับโลกที่ขึ้นอยู่กับประเภทของอาคารของคุณ



### ทำไมฉันต้องกำหนดฐานศูนย์?

เหตุผลหลักในการกำหนดฐานศูนย์คือเพื่อสร้างการวิเคราะห์และดูข้อมูลอย่างง่ายเกี่ยวกับวิธีการบริโภคปัจจุบันของคุณติดตามตามฐานศูนย์ที่กำหนดไว้



### ฐานศูนย์ทำงานอย่างไร?

คุณสามารถกำหนดฐานศูนย์หลายๆ อัน คุณต้องตั้งชื่อฐานศูนย์ของคุณก่อน แล้วจึงกำหนดค่าที่สอดคล้องกับฐานศูนย์ของคุณ

มีกฎบางอย่างเกี่ยวกับวิธีการทำงาน:

1. ฐานศูนย์เดียวสามารถใช้กับหลายประเภทของสาธารณูปโภค ดังนั้นคุณสามารถกำหนดฐานศูนย์ที่เรียกว่า 'ฐานศูนย์ภายในของบริษัทของฉัน' และตั้งค่าฐานศูนย์ที่แตกต่างกันสำหรับการบริโภคน้ำและการบริโภคพลังงานและการบริโภคก๊าซ
2. แต่ละเมตร, เมตรเสมือน, กลุ่ม, แท็กหรือสถานที่สามารถมีค่าฐานศูนย์ที่แนบมากับมัน คุณสามารถกำหนดค่าฐานศูนย์ที่เฉพาะเจาะจงสำหรับแต่ละรายการ ฐานศูนย์อาจแตกต่างกันอย่างมากระหว่างรายการต่างๆ ขึ้นอยู่กับลักษณะของรายการ ตัวอย่างเช่น, เมตรที่บันทึกการใช้ไฟฟ้าทั้งหมดของอาคารจะมีฐานศูนย์ที่แตกต่างจากเมตรที่บันทึกการใช้ปลั๊กในชั้น 3
3. ฐานศูนย์เป็นทางเลือก - คุณสามารถเลือกที่จะไม่กำหนดค่าสำหรับบางหรือทั้งหมดของรายการ
4. ฐานศูนย์ถูกกำหนดเป็นค่าการบริโภครายวัน เมื่อทำการวิเคราะห์, ระบบจะคำนวณว่าฐานศูนย์ควรจะเป็นอย่างไรสำหรับช่วงเวลาที่คุณเลือกขึ้นอยู่กับค่ารายวันที่คุณกำหนด
