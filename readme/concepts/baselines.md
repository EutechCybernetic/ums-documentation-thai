# ฐานอ้างอิง

ฐานอ้างอิงช่วยให้คุณสามารถติดตามการบริโภคจริงของคุณเมื่อเทียบกับการบริโภคที่คุณคิดว่าควรจะเป็น

การบริโภคที่เหมาะสมคืออะไร? มันขึ้นอยู่กับความต้องการของคุณ!

การบริโภคที่เหมาะสมของคุณอาจจะเป็นอะไรก็ได้ที่คุณบันทึกเป็นการบริโภคในจุดเวลาที่เฉพาะเจาะจงในอดีตและคุณต้องการเปรียบเทียบว่าการบริโภคปัจจุบันของคุณติดตามอย่างไร

การบริโภคที่เหมาะสมของคุณอาจจะขึ้นอยู่กับมาตรฐานที่เกี่ยวข้องกับกฎหมายหรือการปฏิบัติที่คุณกำลังมองหาวิธีปฏิบัติตาม

คุณสามารถกำหนดฐานอ้างอิงหลายๆ อัน - บางอันอาจจะขึ้นอยู่กับมาตรฐานภายในของคุณเกี่ยวกับการบริโภคที่ควรจะเป็นอย่างไร และอื่น ๆ อาจจะเพื่อติดตามมาตรฐานของประเทศหรือมาตรฐานทั่วโลกขึ้นอยู่กับประเภทของอาคารของคุณ



### ทำไมฉันต้องกำหนดฐานอ้างอิง?

เหตุผลหลักในการกำหนดฐานอ้างอิงคือเพื่อสร้างการวิเคราะห์และดูข้อมูลได้ง่ายเกี่ยวกับวิธีการบริโภคปัจจุบันของคุณติดตามฐานอ้างอิงที่คงที่



### ฐานอ้างอิงทำงานอย่างไร?

คุณสามารถกำหนดฐานอ้างอิงหลายๆ อัน คุณต้องกำหนดชื่อสำหรับฐานอ้างอิงของคุณก่อน แล้วจึงกำหนดค่าที่สอดคล้องกับฐานอ้างอิงของคุณ

มีกฎบางอย่างเกี่ยวกับวิธีการทำงานของฐานอ้างอิง:

1. ฐานอ้างอิงเดียวสามารถนำไปใช้กับหลายประเภทของสาธารณูปโภค ดังนั้นคุณสามารถกำหนดฐานอ้างอิงที่ชื่อว่า 'ฐานอ้างอิงภายในของบริษัทของฉัน' และตั้งค่าฐานอ้างอิงที่แตกต่างกันสำหรับการบริโภคน้ำและการบริโภคพลังงานและการบริโภคแก๊ส
2. แต่ละเมตร, เมตรเสมือน, กลุ่ม, แท็กหรือสถานที่ สามารถมีค่าฐานอ้างอิงที่แนบมากับมัน คุณสามารถกำหนดค่าฐานอ้างอิงที่เฉพาะเจาะจงสำหรับแต่ละรายการ ฐานอ้างอิงอาจจะแตกต่างกันอย่างมากระหว่างรายการต่างๆ ขึ้นอยู่กับลักษณะของรายการ ตัวอย่างเช่น, เมตรที่บันทึกการใช้ไฟฟ้าทั้งหมดของอาคารจะมีฐานอ้างอิงที่แตกต่างจากเมตรที่บันทึกการใช้ปลั๊กในชั้น 3
3. ฐานอ้างอิงเป็นทางเลือก - คุณสามารถเลือกไม่กำหนดค่าสำหรับบางรายการหรือทุกรายการ
4. ฐานอ้างอิงถูกกำหนดเป็นค่าการบริโภครายวัน เมื่อทำการวิเคราะห์, ระบบจะคำนวณว่าฐานอ้างอิงควรจะเป็นเท่าไหร่สำหรับช่วงเวลาที่คุณเลือกขึ้นอยู่กับค่ารายวันที่คุณกำหนด
