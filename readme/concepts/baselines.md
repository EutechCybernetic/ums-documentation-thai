# ฐานศูนย์

ฐานศูนย์ช่วยให้คุณสามารถติดตามการบริโภคจริงของคุณเมื่อเทียบกับการบริโภคที่เหมาะสม

การบริโภคที่เหมาะสมคืออะไร? มันขึ้นอยู่กับความต้องการของคุณ!

การบริโภคที่เหมาะสมของคุณอาจจะเป็นสิ่งที่คุณบันทึกไว้เป็นการบริโภคในจุดเวลาที่เฉพาะเจาะจงในอดีต และคุณต้องการเปรียบเทียบว่าการบริโภคปัจจุบันของคุณติดตามอย่างไร

การบริโภคที่เหมาะสมของคุณอาจจะขึ้นอยู่กับมาตรฐานที่มีอยู่ตามกฎหมายหรือมาตรฐานการปฏิบัติที่คุณกำลังพยายามปฏิบัติตาม

คุณสามารถกำหนดฐานศูนย์หลายๆ อัน - บางอันอาจจะขึ้นอยู่กับมาตรฐานภายในของคุณเกี่ยวกับการบริโภคที่ควรจะเป็นอย่างไร และอื่นๆ อาจจะเพื่อติดตามตามมาตรฐานของประเทศหรือมาตรฐานสากลขึ้นอยู่กับประเภทของอาคารของคุณ



### ทำไมฉันต้องกำหนดฐานศูนย์?

เหตุผลหลักในการกำหนดฐานศูนย์คือเพื่อสร้างการวิเคราะห์และดูข้อมูลได้ง่ายเกี่ยวกับวิธีการบริโภคปัจจุบันของคุณติดตามฐานศูนย์ที่กำหนดไว้



### ฐานศูนย์ทำงานอย่างไร?

คุณสามารถกำหนดฐานศูนย์หลายๆ อัน คุณต้องตั้งชื่อฐานศูนย์ของคุณก่อน แล้วจึงกำหนดค่าที่สอดคล้องกับฐานศูนย์ของคุณ

มีกฎบางอย่างเกี่ยวกับวิธีการทำงานของฐานศูนย์:

1. ฐานศูนย์เดียวสามารถใช้กับหลายประเภทของสาธารณูปโภค ดังนั้นคุณสามารถกำหนดฐานศูนย์ที่เรียกว่า 'ฐานศูนย์ภายในของบริษัทของฉัน' และตั้งค่าฐานศูนย์ที่แตกต่างกันสำหรับการบริโภคน้ำ การบริโภคพลังงาน และการบริโภคแก๊ส
2. แต่ละเมตร, เมตรเสมือนจริง, กลุ่ม, แท็ก หรือสถานที่สามารถมีค่าฐานศูนย์ที่แนบมากับมัน คุณสามารถกำหนดค่าฐานศูนย์ที่เฉพาะเจาะจงสำหรับแต่ละอัน ฐานศูนย์อาจจะแตกต่างกันอย่างมากระหว่างอันตรายขึ้นอยู่กับลักษณะของรายการ ตัวอย่างเช่น, เมตรที่บันทึกการใช้ไฟฟ้าทั่วทั้งอาคารจะมีฐานศูนย์ที่แตกต่างจากเมตรที่บันทึกการใช้ไฟฟ้าที่ชั้น 3
3. ฐานศูนย์เป็นสิ่งที่ไม่จำเป็น - คุณสามารถเลือกไม่ต้องกำหนดค่าสำหรับบางอันหรือทั้งหมด
4. ฐานศูนย์ถูกกำหนดเป็นค่าการบริโภครายวัน เมื่อทำการวิเคราะห์, ระบบจะคำนวณว่าฐานศูนย์ควรจะเป็นอย่างไรสำหรับช่วงเวลาที่คุณเลือกขึ้นอยู่กับค่ารายวันที่คุณกำหนด
