# ฐานศูนย์

ฐานศูนย์ช่วยให้คุณสามารถติดตามการบริโภคจริงของคุณเมื่อเทียบกับการบริโภคที่คุณคิดว่าควรจะเป็น

การบริโภคที่เหมาะสมคืออะไร? มันขึ้นอยู่กับความต้องการของคุณ!

การบริโภคที่เหมาะสมของคุณอาจจะเป็นอะไรก็ได้ที่คุณได้บันทึกเป็นการบริโภคในจุดเวลาที่เฉพาะเจาะจงในอดีต และคุณต้องการเปรียบเทียบว่าการบริโภคปัจจุบันของคุณเปรียบเทียบกับสิ่งนั้นอย่างไร

การบริโภคที่เหมาะสมของคุณอาจจะขึ้นอยู่กับมาตรฐานที่เกี่ยวข้องกับการปฏิบัติหรือการปฏิบัติตามกฎหมายที่คุณกำลังมองหาวิธีที่จะปฏิบัติตาม

คุณสามารถกำหนดฐานศูนย์หลายๆ อัน - บางอันอาจจะขึ้นอยู่กับมาตรฐานภายในของคุณเกี่ยวกับการบริโภคควรจะเป็นอย่างไร และอื่น ๆ อาจจะเพื่อติดตามตามมาตรฐานของประเทศหรือมาตรฐานทั่วโลกขึ้นอยู่กับประเภทของอาคารของคุณ



### ทำไมฉันต้องกำหนดฐานศูนย์?

เหตุผลหลักในการกำหนดฐานศูนย์คือเพื่อสร้างการวิเคราะห์และดูข้อมูลได้ง่ายเกี่ยวกับวิธีการบริโภคปัจจุบันของคุณเปรียบเทียบกับฐานศูนย์ที่คงที่



### ฐานศูนย์ทำงานอย่างไร?

คุณสามารถกำหนดฐานศูนย์หลายๆ อัน คุณต้องกำหนดชื่อสำหรับฐานศูนย์ของคุณก่อน แล้วจึงกำหนดค่าที่สอดคล้องกับฐานศูนย์ของคุณ

มีกฎบางอย่างเกี่ยวกับวิธีการทำงานของฐานศูนย์:

1. ฐานศูนย์เดียวสามารถใช้กับประเภทสาธารณูปโภคหลายประเภท ดังนั้นคุณสามารถกำหนดฐานศูนย์ที่เรียกว่า 'ฐานศูนย์ภายในของบริษัทของฉัน' และตั้งค่าฐานศูนย์ที่แตกต่างกันสำหรับการบริโภคน้ำ การบริโภคพลังงาน และการบริโภคก๊าซ
2. แต่ละเมตร, เมตรเสมือน, กลุ่ม, แท็ก หรือสถานที่ สามารถมีค่าฐานศูนย์ที่แนบมากับมัน คุณสามารถกำหนดค่าฐานศูนย์ที่เฉพาะเจาะจงสำหรับแต่ละรายการ ฐานศูนย์อาจจะแตกต่างกันอย่างมากระหว่างรายการต่างๆ ขึ้นอยู่กับลักษณะของรายการ ตัวอย่างเช่น เมตรที่บันทึกการใช้ไฟทั้งหมดของอาคารจะมีฐานศูนย์ที่แตกต่างจากเมตรที่บันทึกการใช้ปลั๊กในชั้น 3
3. ฐานศูนย์เป็นสิ่งที่ไม่จำเป็น - คุณสามารถเลือกที่จะไม่กำหนดค่าสำหรับบางรายการหรือทั้งหมด
4. ฐานศูนย์ถูกกำหนดเป็นค่าการบริโภครายวัน เมื่อทำการวิเคราะห์ ระบบจะคำนวณว่าฐานศูนย์ควรจะเป็นอย่างไรสำหรับช่วงเวลาที่คุณเลือกขึ้นอยู่กับค่ารายวันที่คุณกำหนด
