# ฐานศูนย์

ฐานศูนย์ช่วยให้คุณสามารถติดตามการบริโภคจริงของคุณเมื่อเทียบกับการบริโภคที่เป็นไอเดีย

การบริโภคที่เหมาะสมคืออะไร? มันขึ้นอยู่กับความต้องการของคุณ!

การบริโภคที่เหมาะสมของคุณอาจเป็นสิ่งที่คุณบันทึกไว้เป็นการบริโภคในจุดเวลาที่เฉพาะเจาะจงในอดีตและคุณต้องการเปรียบเทียบว่าการบริโภคปัจจุบันของคุณติดตามอย่างไร

การบริโภคที่เหมาะสมของคุณอาจจะขึ้นอยู่กับมาตรฐานที่เกี่ยวข้องกับกฎหมายหรือการปฏิบัติที่คุณกำลังมองหาวิธีปฏิบัติตาม

คุณสามารถกำหนดฐานศูนย์หลายๆ อัน - บางอันอาจจะขึ้นอยู่กับมาตรฐานภายในของคุณเกี่ยวกับการบริโภคที่ควรจะเป็นอย่างไรและอื่น ๆ อาจจะเพื่อติดตามตามมาตรฐานของประเทศหรือมาตรฐานทั่วโลกขึ้นอยู่กับประเภทของอาคารของคุณ



### ทำไมฉันต้องกำหนดฐานศูนย์?

เหตุผลหลักในการกำหนดฐานศูนย์คือเพื่อสร้างการวิเคราะห์และดูข้อมูลอย่างง่ายเกี่ยวกับการบริโภคปัจจุบันของคุณติดตามฐานศูนย์ที่คงที่



### ฐานศูนย์ทำงานอย่างไร?

คุณสามารถกำหนดฐานศูนย์หลายๆ อัน คุณต้องกำหนดชื่อสำหรับฐานศูนย์ของคุณก่อน แล้วจึงกำหนดค่าที่สอดคล้องกับฐานศูนย์ของคุณ

มีกฎบางอย่างเกี่ยวกับวิธีการทำงาน:

1. ฐานศูนย์เดียวสามารถใช้กับหลายประเภทของสาธารณูปโภค ดังนั้นคุณสามารถกำหนดฐานศูนย์ที่เรียกว่า 'ฐานศูนย์ภายในของบริษัทของฉัน' และตั้งค่าฐานศูนย์ที่แตกต่างกันสำหรับการบริโภคน้ำและการบริโภคพลังงานและการบริโภคก๊าซ
2. แต่ละเมตร, เมตรเสมือน, กลุ่ม, แท็กหรือสถานที่สามารถมีค่าฐานศูนย์ที่แนบมากับมัน คุณสามารถกำหนดค่าฐานศูนย์ที่เฉพาะเจาะจงสำหรับแต่ละรายการ ฐานศูนย์อาจแตกต่างกันอย่างมากระหว่างรายการต่างๆ ขึ้นอยู่กับลักษณะของรายการ ตัวอย่างเช่น, เมตรที่บันทึกการใช้ไฟทั่วอาคารทั้งหมดจะมีฐานศูนย์ที่แตกต่างจากเมตรที่บันทึกการใช้ปลั๊กในชั้น 3
3. ฐานศูนย์เป็นทางเลือก - คุณสามารถเลือกที่จะไม่กำหนดค่าสำหรับบางหรือทั้งหมดของรายการ
4. ฐานศูนย์ถูกกำหนดเป็นค่าการบริโภครายวัน เมื่อทำการวิเคราะห์ระบบจะคำนวณว่าฐานศูนย์ควรจะเป็นอย่างไรสำหรับช่วงเวลาที่คุณเลือกขึ้นอยู่กับค่ารายวันที่คุณกำหนด
