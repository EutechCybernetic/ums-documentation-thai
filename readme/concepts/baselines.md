# ฐานศูนย์

ฐานศูนย์ช่วยให้คุณสามารถติดตามการบริโภคจริงของคุณเมื่อเทียบกับการบริโภคที่เป็นไอเดีย

การบริโภคที่เหมาะสมคืออะไร? มันขึ้นอยู่กับความต้องการของคุณ!

การบริโภคที่เหมาะสมของคุณอาจจะเป็นสิ่งที่คุณบันทึกเป็นการบริโภคในจุดเวลาที่เฉพาะเจาะจงในอดีตและคุณต้องการเปรียบเทียบว่าการบริโภคปัจจุบันของคุณติดตามตามนั้นอย่างไร

การบริโภคที่เหมาะสมของคุณอาจจะขึ้นอยู่กับมาตรฐานที่ตามกฎหมายหรือมาตรฐานการปฏิบัติที่คุณกำลังมองหาเพื่อปฏิบัติตาม

คุณสามารถกำหนดฐานศูนย์หลายรายการ - บางรายการอาจจะขึ้นอยู่กับมาตรฐานภายในของคุณเกี่ยวกับการบริโภคที่ควรจะเป็นอย่างไร และอื่น ๆ อาจจะเพื่อติดตามตามมาตรฐานของประเทศหรือมาตรฐานทั่วโลกขึ้นอยู่กับประเภทของอาคารของคุณ



### ทำไมฉันต้องกำหนดฐานศูนย์?

เหตุผลหลักในการกำหนดฐานศูนย์คือเพื่อสร้างการวิเคราะห์และดูข้อมูลได้อย่างง่ายดายเกี่ยวกับวิธีการบริโภคปัจจุบันของคุณติดตามตามฐานศูนย์ที่กำหนดไว้



### ฐานศูนย์ทำงานอย่างไร?

คุณสามารถกำหนดฐานศูนย์หลายรายการ คุณต้องกำหนดชื่อให้กับฐานศูนย์ของคุณก่อน แล้วจึงกำหนดค่าที่สอดคล้องกับฐานศูนย์ของคุณ

มีกฎบางประการสำหรับวิธีการทำงานของฐานศูนย์:

1. ฐานศูนย์เดียวสามารถนำไปใช้กับหลายประเภทของสาธารณูปโภค ดังนั้นคุณสามารถกำหนดฐานศูนย์ที่เรียกว่า 'ฐานศูนย์ภายในของบริษัทของฉัน' และตั้งค่าฐานศูนย์ที่แตกต่างกันสำหรับการบริโภคน้ำและการบริโภคพลังงานและการบริโภคก๊าซ
2. แต่ละเมตร, เมตรเสมือน, กลุ่ม, แท็กหรือสถานที่สามารถมีค่าฐานศูนย์ที่แนบมากับมัน คุณสามารถกำหนดค่าฐานศูนย์ที่เฉพาะเจาะจงสำหรับแต่ละรายการ ฐานศูนย์อาจจะแตกต่างกันอย่างมากระหว่างรายการต่างๆ ขึ้นอยู่กับลักษณะของรายการ ตัวอย่างเช่น, เมตรที่บันทึกการใช้ไฟฟ้าทั่วทั้งอาคารจะมีฐานศูนย์ที่แตกต่างจากเมตรที่บันทึกการใช้ปลั๊กในชั้น 3
3. ฐานศูนย์เป็นทางเลือก - คุณสามารถเลือกไม่ต้องกำหนดค่าสำหรับบางรายการหรือรายการทั้งหมด
4. ฐานศูนย์ถูกกำหนดเป็นค่าการบริโภครายวัน เมื่อทำการวิเคราะห์, ระบบจะคำนวณว่าฐานศูนย์ควรจะเป็นอย่างไรสำหรับช่วงวันที่คุณเลือกขึ้นอยู่กับค่ารายวันที่คุณกำหนด
