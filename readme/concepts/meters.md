# เมตร

เมตรทางฟิสิกส์เป็นสิ่งที่บันทึกการบริโภค ดังชื่อที่แนะนำ คุณจะทำการแมปเหล่านี้ไปยังเมตรทางฟิสิกส์ในโลกจริงที่คุณมี การอ่านจะถูกบันทึกต่อเมตรในช่วงเวลาที่สม่ำเสมอและเหล่านั้นคือจุดข้อมูลดิบที่ขับเคลื่อนการวิเคราะห์ทั้งหมด

ถ้าอาคารของคุณมีเมตรพลังงาน 3 เมตรและเมตรน้ำ 1 เมตร คุณจะลงทะเบียนเมตรพลังงาน 3 เมตรและเมตรน้ำ 1 เมตร

แต่ละเมตรมีคุณสมบัติบางอย่างที่สามารถกำหนดค่าได้:

1. ประเภทยูทิลิตี้ (มันเป็นเมตรพลังงานหรือไม่? หรือเมตรน้ำ? หรืออย่างอื่น?)
2. ชื่อ - สามารถเป็นอะไรก็ได้ที่อธิบายได้
3. หน่วย - เมตรนี้ใช้หน่วยอะไรในการบันทึกและแสดงการอ่าน
4. สถานที่ - เมตรนี้เชื่อมโยงกับสถานที่อะไร เมตรสามารถเชื่อมโยงกับหน่วยรายบุคคล ชั้นหรืออาคารทั้งหมด คุณจะเลือกสถานที่ที่ได้รับบริการจากเมตรนี้
5. แท็ก - คุณสามารถแท็กเมตรด้วยแท็กที่คุณกำหนดไว้ล่วงหน้า ดูส่วนของการแท็กเพื่อข้อมูลเพิ่มเติมเกี่ยวกับทำไมคุณต้องการทำเช่นนั้น
6. การกำหนดค่าเบสไลน์ - กำหนดค่าเบสไลน์ทุกวันสำหรับแต่ละเมตร - ดูส่วนเบสไลน์สำหรับข้อมูลเพิ่มเติม

คุณสามารถบันทึกการอ่านสำหรับเมตรในหลายวิธีรวมถึงการป้อนการอ่านด้วยตนเอง อัปโหลดไฟล์ csv อัปโหลดบิลหรือการรวมกับระบบ IoT และ BMS
