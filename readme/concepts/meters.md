# เมตร

เมตรทางฟิสิกส์คือสิ่งที่บันทึกการใช้งาน ดังที่ชื่อเสนอ คุณจะทำการแมปเมตรเหล่านี้กับเมตรทางฟิสิกส์ในโลกจริงที่คุณมี การอ่านค่าจะถูกบันทึกตามเมตรในช่วงเวลาที่กำหนด และเป็นจุดข้อมูลดิบที่ขับเคลื่อนการวิเคราะห์ทั้งหมด

ถ้าอาคารของคุณมีเมตรพลังงาน 3 เมตร และเมตรน้ำ 1 เมตร คุณจะลงทะเบียน 3 เมตรเป็นเมตรพลังงาน และ 1 เมตรเป็นเมตรน้ำ

แต่ละเมตรมีคุณสมบัติบางอย่างที่สามารถกำหนดค่าได้:

1. ประเภทของสาธารณูปโภค (มันเป็นเมตรพลังงานหรือไม่? หรือเมตรน้ำ? หรืออย่างอื่น?)
2. ชื่อ - สามารถเป็นอะไรก็ได้ที่อธิบายได้
3. หน่วย - หน่วยที่เมตรนี้ใช้สำหรับการบันทึกและแสดงการอ่านค่า
4. สถานที่ - สถานที่ที่เมตรนี้เชื่อมโยง สามารถเชื่อมโยงเมตรกับหน่วยเดี่ยวๆ ชั้น หรืออาคารทั้งหมด คุณจะเลือกสถานที่ที่ได้รับบริการจากเมตรนี้
5. แท็ก - คุณสามารถแท็กเมตรด้วยแท็กที่คุณกำหนดไว้ล่วงหน้า ดูส่วนเกี่ยวกับการแท็กเพื่อข้อมูลเพิ่มเติมเกี่ยวกับเหตุผลที่คุณต้องการทำเช่นนั้น
6. การกำหนดค่าเบสไลน์ - กำหนดค่าเบสไลน์ประจำวันสำหรับแต่ละเมตร - ดูส่วนของเบสไลน์เพื่อข้อมูลเพิ่มเติม

คุณสามารถบันทึกการอ่านค่าสำหรับเมตรได้หลายวิธี รวมถึงการป้อนค่าอ่านด้วยตนเอง อัปโหลดไฟล์ csv อัปโหลดบิล หรือการรวมกับระบบ IoT และ BMS
