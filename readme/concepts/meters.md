# เมตร

เมตรทางฟิสิกส์คือสิ่งที่บันทึกการบริโภค ดังชื่อที่แนะนำ คุณจะทำการแมปเมตรเหล่านี้กับเมตรทางฟิสิกส์ในโลกจริงที่คุณมี การอ่านค่าจะถูกบันทึกตามเมตรในช่วงเวลาที่สม่ำเสมอและเหล่านั้นคือจุดข้อมูลดิบที่ขับเคลื่อนการวิเคราะห์ทั้งหมด

หากอาคารของคุณมีเมตรพลังงาน 3 เมตรและเมตรน้ำ 1 เมตร คุณจะลงทะเบียนเมตรพลังงาน 3 เมตรและเมตรน้ำ 1 เมตร

แต่ละเมตรมีคุณสมบัติบางอย่างที่สามารถกำหนดค่าได้:

1. ประเภทยูทิลิตี้ (มันเป็นเมตรพลังงานหรือไม่? หรือเมตรน้ำ? หรืออย่างอื่น?)
2. ชื่อ - สามารถเป็นอะไรก็ได้ที่อธิบายได้
3. หน่วย - เมตรนี้ใช้หน่วยอะไรในการบันทึกและแสดงการอ่านค่า
4. สถานที่ - เมตรนี้เชื่อมโยงกับสถานที่อย่างไร เมตรสามารถเชื่อมโยงกับหน่วยเดี่ยวๆ, ชั้นหรืออาคารทั้งหมด คุณจะเลือกสถานที่ที่ได้รับการ _บริการ_ จากเมตรนี้
5. แท็ก - คุณสามารถแท็กเมตรด้วยแท็กที่คุณกำหนดไว้ล่วงหน้า ดูส่วนของการแท็กสำหรับข้อมูลเพิ่มเติมเกี่ยวกับทำไมคุณต้องการทำเช่นนั้น
6. การกำหนดค่าเบสไลน์ - กำหนดค่าเบสไลน์ประจำวันสำหรับแต่ละเมตร - ดูส่วนเบสไลน์สำหรับข้อมูลเพิ่มเติม

คุณสามารถบันทึกการอ่านค่าสำหรับเมตรในหลายวิธี รวมถึงการป้อนค่าอ่านด้วยตนเอง, อัปโหลดไฟล์ csv, อัปโหลดบิลหรือการรวมกับระบบ IoT และ BMS
