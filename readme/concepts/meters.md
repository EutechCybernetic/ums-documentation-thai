# เมตร

เมตรทางฟิสิกส์เป็นสิ่งที่บันทึกการใช้งาน ดังชื่อที่แนะนำ คุณจะทำการแมปเมตรเหล่านี้กับเมตรทางฟิสิกส์ในโลกจริงที่คุณมี การอ่านค่าจะถูกบันทึกตามเมตรในช่วงเวลาที่ปกติ และนั่นคือจุดข้อมูลดิบที่ขับเคลื่อนการวิเคราะห์ทั้งหมด

หากอาคารของคุณมีเมตรพลังงาน 3 เมตรและเมตรน้ำ 1 เมตร คุณจะลงทะเบียนเมตรพลังงาน 3 เมตรและเมตรน้ำ 1 เมตร

แต่ละเมตรมีคุณสมบัติบางอย่างที่สามารถกำหนดค่าได้:

1. ประเภทของยูทิลิตี้ (มันเป็นเมตรพลังงานหรือไม่? หรือเมตรน้ำ? หรืออย่างอื่น?)
2. ชื่อ - สามารถเป็นอะไรก็ได้ที่อธิบายได้
3. หน่วย - หน่วยที่เมตรนี้ใช้สำหรับการบันทึกและแสดงการอ่านค่า
4. สถานที่ - สถานที่ที่เมตรนี้เชื่อมโยงอยู่ เมตรสามารถเชื่อมโยงกับยูนิตแต่ละยูนิต, ชั้นหรืออาคารทั้งหมด คุณจะเลือกสถานที่ที่ได้รับการ _บริการ_ จากเมตรนี้
5. แท็ก - คุณสามารถแท็กเมตรด้วยแท็กที่คุณกำหนดไว้ล่วงหน้า ดูส่วนของการแท็กสำหรับข้อมูลเพิ่มเติมเกี่ยวกับเหตุผลที่คุณต้องการทำเช่นนั้น
6. การกำหนดค่าเบสไลน์ - กำหนดค่าเบสไลน์ประจำวันสำหรับแต่ละเมตร - ดูส่วนเบสไลน์สำหรับข้อมูลเพิ่มเติม

คุณสามารถบันทึกการอ่านค่าสำหรับเมตรในหลายวิธี รวมถึงการป้อนค่าอ่านค่าด้วยตนเอง, อัปโหลดไฟล์ csv, อัปโหลดบิลหรือการรวมกับระบบ IoT และ BMS
