# เมตร

เมตรทางฟิสิกส์เป็นสิ่งที่บันทึกการบริโภค ดังชื่อที่แนะนำ คุณจะทำการแมปเหล่านี้กับเมตรทางฟิสิกส์ในโลกจริงที่คุณมี การอ่านจะถูกบันทึกต่อเนื่องตามเมตรที่คุณมี และเหล่านั้นคือจุดข้อมูลดิบที่ขับเคลื่อนการวิเคราะห์ทั้งหมด

หากอาคารของคุณมีเมตรพลังงาน 3 เครื่องและเมตรน้ำ 1 เครื่อง คุณจะลงทะเบียนเมตรพลังงาน 3 เครื่องและเมตรน้ำ 1 เครื่อง

แต่ละเมตรมีคุณสมบัติบางอย่างที่สามารถกำหนดค่าได้:

1. ประเภทยูทิลิตี้ (มันเป็นเมตรพลังงานหรือไม่? หรือเมตรน้ำ? หรืออย่างอื่น?)
2. ชื่อ - สามารถเป็นอะไรก็ได้ที่อธิบายได้
3. หน่วย - หน่วยที่เมตรนี้ใช้สำหรับการบันทึกและแสดงการอ่าน
4. สถานที่ - สถานที่ที่เมตรนี้เชื่อมต่ออยู่ เมตรสามารถเชื่อมต่อกับหน่วยรายบุคคล ชั้นหรืออาคารทั้งหมด คุณจะเลือกสถานที่ที่ได้รับบริการจากเมตรนี้
5. แท็ก - คุณสามารถแท็กเมตรด้วยแท็กที่คุณกำหนดไว้ล่วงหน้า ดูส่วนของการแท็กเพื่อข้อมูลเพิ่มเติมเกี่ยวกับทำไมคุณต้องการทำเช่นนั้น
6. การกำหนดค่าระดับฐาน - กำหนดค่าระดับฐานรายวันสำหรับแต่ละเมตร - ดูส่วนของระดับฐานสำหรับข้อมูลเพิ่มเติม

คุณสามารถบันทึกการอ่านสำหรับเมตรได้หลายวิธี รวมถึงการป้อนการอ่านด้วยตนเอง อัปโหลดไฟล์ csv อัปโหลดบิลหรือการรวมกับระบบ IoT และ BMS
