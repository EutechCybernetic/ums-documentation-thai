# เมตร

เมตรทางฟิสิกส์เป็นสิ่งที่บันทึกการบริโภค ดังชื่อที่แนะนำ คุณจะทำการแมปเมตรเหล่านี้กับเมตรทางฟิสิกส์ในโลกจริงที่คุณมี การอ่านค่าจะถูกบันทึกต่อเมตรในช่วงเวลาที่สม่ำเสมอและเป็นจุดข้อมูลดิบที่ขับเคลื่อนการวิเคราะห์ทั้งหมด

หากอาคารของคุณมีเมตรพลังงาน 3 เมตรและเมตรน้ำ 1 เมตร คุณจะลงทะเบียนเมตรพลังงาน 3 เมตรและเมตรน้ำ 1 เมตร

แต่ละเมตรมีคุณสมบัติบางอย่างที่สามารถกำหนดค่าได้:

1. ประเภทยูทิลิตี้ (มันเป็นเมตรพลังงานหรือไม่? หรือเมตรน้ำ? หรืออย่างอื่น?)
2. ชื่อ - สามารถเป็นอะไรก็ได้ที่อธิบายได้
3. หน่วย - หน่วยที่เมตรนี้ใช้สำหรับการบันทึกและแสดงการอ่านค่า
4. สถานที่ - สถานที่ที่เมตรนี้ถูกผูกมัดไว้ เมตรสามารถผูกมัดไว้กับหน่วยแต่ละหน่วย, ชั้นหรืออาคารทั้งหมด คุณจะเลือกสถานที่ที่ได้รับบริการจากเมตรนี้
5. แท็ก - คุณสามารถแท็กเมตรด้วยแท็กที่คุณกำหนดไว้ล่วงหน้า ดูส่วนของการแท็กเพื่อข้อมูลเพิ่มเติมเกี่ยวกับทำไมคุณต้องการทำเช่นนั้น
6. การกำหนดค่าเบสไลน์ - กำหนดค่าเบสไลน์ประจำวันสำหรับแต่ละเมตร - ดูส่วนเบสไลน์สำหรับข้อมูลเพิ่มเติม

คุณสามารถบันทึกการอ่านค่าสำหรับเมตรได้หลายวิธี รวมถึงการป้อนค่าอ่านด้วยตนเอง, อัปโหลดไฟล์ csv, อัปโหลดใบแจ้งหนี้หรือการรวมกับระบบ IoT และ BMS
