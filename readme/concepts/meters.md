# เมตร

เมตรทางฟิสิกส์เป็นสิ่งที่บันทึกการบริโภค ดังชื่อที่แนะนำ คุณจะทำการแมปเหล่านี้ไปยังเมตรทางฟิสิกส์ในโลกจริงที่คุณมี การอ่านจะถูกบันทึกต่อเนื่องตามเมตรที่กำหนด และเหล่านี้คือจุดข้อมูลดิบที่ขับเคลื่อนการวิเคราะห์ทั้งหมด

หากอาคารของคุณมีเมตรพลังงาน 3 เมตรและเมตรน้ำ 1 เมตร คุณจะลงทะเบียนเมตรพลังงาน 3 เมตรและเมตรน้ำ 1 เมตร

แต่ละเมตรมีคุณสมบัติบางอย่างที่สามารถกำหนดค่าได้:

1. ประเภทยูทิลิตี้ (มันเป็นเมตรพลังงานหรือไม่? หรือเมตรน้ำ? หรืออย่างอื่น?)
2. ชื่อ - สามารถเป็นอะไรก็ได้ที่อธิบายได้
3. หน่วย - หน่วยที่เมตรนี้ใช้สำหรับการบันทึกและแสดงการอ่าน
4. สถานที่ - สถานที่ที่เมตรนี้เกี่ยวข้อง สามารถเชื่อมโยงเมตรกับหน่วยรายบุคคล ชั้นหรืออาคารทั้งหมด คุณจะเลือกสถานที่ที่ถูก _บริการ_ โดยเมตรนี้
5. แท็ก - คุณสามารถแท็กเมตรด้วยแท็กที่คุณกำหนดเอง ดูส่วนเกี่ยวกับการแท็กสำหรับข้อมูลเพิ่มเติมเกี่ยวกับเหตุผลที่คุณต้องการทำเช่นนั้น
6. การกำหนดค่าเบสไลน์ - กำหนดค่าเบสไลน์ประจำวันสำหรับแต่ละเมตร - ดูส่วนเบสไลน์สำหรับข้อมูลเพิ่มเติม

คุณสามารถบันทึกการอ่านสำหรับเมตรได้หลายวิธี รวมถึงการป้อนการอ่านด้วยตนเอง อัปโหลดไฟล์ csv อัปโหลดบิลหรือการรวมกับระบบ IoT และ BMS
