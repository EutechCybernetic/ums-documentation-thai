# การตั้งค่ามิเตอร์ใหม่

สำหรับมิเตอร์ที่บันทึกการอ่านค่าสะสม, การตั้งค่ามิเตอร์ใหม่จะเกิดขึ้นโดยอัตโนมัติ

หากการอ่านค่าล่าสุดน้อยกว่าค่าก่อนหน้านี้, จะมีช่วงว่างชั่วคราวในค่าพลังงานณ เวลาที่ตั้งค่าใหม่

ต่อมา, เมื่อการอ่านค่าถัดไปเข้ามา - ค่านี้จะถูกคำนวณขึ้นอยู่กับการอ่านค่าก่อนหน้าที่ถูกส่ง



โปรดทราบว่า การอ่านค่า '0' (ศูนย์) จะถูกละทิ้งโดยไม่เป็นที่สนใจและถือว่าเป็นข้อผิดพลาดในการสื่อสารหรือข้อผิดพลาดในการรวมข้อมูล ดังนั้นหากมีการบันทึกการอ่านค่าณ เวลาที่ตั้งค่าใหม่, ค่านี้จะไม่ถูกใช้จนกว่าการอ่านค่าถัดไปจะถูกส่ง
