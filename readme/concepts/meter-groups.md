# กลุ่มมิเตอร์

ในขณะที่มิเตอร์เสมือนจริงช่วยให้คุณกำหนดมิเตอร์อ่อนที่มีค่าที่คำนวณได้จากมิเตอร์ทางกายภาพหนึ่งหรือมากกว่านั้น กลุ่มมิเตอร์ช่วยให้คุณกำหนดการจัดกลุ่มที่ซับซ้อนขึ้น กลุ่มมิเตอร์สามารถประกอบด้วยมิเตอร์ทางกายภาพหลายตัว, มิเตอร์เสมือนจริง และกลุ่มมิเตอร์อื่น ๆ ด้วย และแต่ละรายการที่เพิ่มเข้าไปในกลุ่มสามารถมีตัวคูณที่เชื่อมโยงอยู่

เช่นเดียวกับมิเตอร์เสมือนจริง, กลุ่มมิเตอร์ยังคงติดตามการบริโภคในรูปแบบการคำนวณของมิเตอร์อื่น ๆ แต่ไม่เหมือนกับมิเตอร์เสมือนจริง, กลุ่มมิเตอร์ยังสามารถอ้างอิงไปยังมิเตอร์เสมือนจริงอื่น ๆ และกลุ่มมิเตอร์อื่น ๆ

เช่นเดียวกับมิเตอร์เสมือนจริง, กลุ่มมิเตอร์ไม่มีหน่วยที่เชื่อมโยงและการบริโภคจะถูกคำนวณเมื่อมิเตอร์ทางกายภาพรับข้อมูลการบริโภค.
