# กลุ่มมิเตอร์

ในขณะที่มิเตอร์เสมือนจริงช่วยให้คุณสามารถกำหนดมิเตอร์อ่อนที่มีค่าที่คำนวณขึ้นมาจากมิเตอร์ทางกายภาพหนึ่งหรือมากกว่านั้น, กลุ่มมิเตอร์ช่วยให้คุณสามารถกำหนดการจัดกลุ่มที่ซับซ้อนมากขึ้น กลุ่มมิเตอร์สามารถประกอบด้วยมิเตอร์ทางกายภาพหลายตัว, มิเตอร์เสมือนจริง และกลุ่มมิเตอร์อื่น ๆ ด้วย และแต่ละรายการที่เพิ่มเข้าไปในกลุ่มสามารถมีปัจจัยตัวคูณที่เชื่อมโยงอยู่

เหมือนกับมิเตอร์เสมือนจริง, กลุ่มมิเตอร์ยังเก็บรายการการบริโภคในรูปแบบการคำนวณของมิเตอร์อื่น ๆ อยู่ แต่ไม่เหมือนกับมิเตอร์เสมือนจริง, กลุ่มมิเตอร์ยังสามารถอ้างอิงถึงมิเตอร์เสมือนจริงอื่น ๆ และกลุ่มมิเตอร์อื่น ๆ ได้

เหมือนกับมิเตอร์เสมือนจริง, กลุ่มมิเตอร์ไม่มีหน่วยที่เชื่อมโยงและการบริโภคจะถูกคำนวณเมื่อมิเตอร์ทางกายภาพที่อยู่ภายใต้รับข้อมูลการบริโภค
