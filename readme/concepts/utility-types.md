# ประเภทของสิ่งอำนวยความสะดวก

ประเภทของสิ่งอำนวยความสะดวกกำหนดประเภทของสิ่งอำนวยความสะดวกที่คุณต้องการวัดและวิเคราะห์ ซึ่งปกติจะมี:

* ไฟฟ้า
* น้ำ
* ก๊าซ
* น้ำเย็น
* ขยะ ฯลฯ...

คุณสามารถเพิ่มประเภทของสิ่งอำนวยความสะดวกเหล่านี้ลงในระบบ ประเภทของสิ่งอำนวยความสะดวกแต่ละประเภทสามารถวัดได้ในหน่วยที่แตกต่างกัน คุณสามารถกำหนดหน่วยที่แตกต่างกันสำหรับประเภทของสิ่งอำนวยความสะดวกแต่ละประเภท ตัวอย่างเช่น ไฟฟ้าสามารถบันทึกได้ในหน่วย kWh หรือ mWh หรือ BTU

#### หน่วย

ภายในประเภทของสิ่งอำนวยความสะดวกแต่ละประเภท คุณต้องกำหนดหน่วยที่เป็นไปได้ที่พวกเขาสามารถวัดได้ สำหรับแต่ละหน่วย - คุณต้องให้ชื่อและปัจจัยการแปลง ปัจจัยการแปลงคือตัวคูณเพื่อแปลงหน่วยนั้นเป็นหน่วยฐาน

คุณสามารถกำหนดหน่วยฐานโดยการให้ปัจจัยการแปลงเป็นหนึ่ง

ตัวอย่างเช่น ถ้าคุณกำลังจัดเก็บปริมาณน้ำ คุณอาจต้องการกำหนดหน่วยต่อไปนี้:

* ลิตร
* แกลลอนสหรัฐ
* แกลลอนอิมพีเรียล

ถ้าเราเลือกลิตรเป็นหน่วยฐาน แล้วสามารถใช้ปัจจัยการแปลงต่อไปนี้:

* ลิตร - 1
* แกลลอนสหรัฐ - 3.785
* แกลลอนอิมพีเรียล - 4.546

ถ้าเราต้องการเลือกแกลลอนสหรัฐเป็นหน่วยฐาน สามารถใช้ปัจจัยการแปลงต่อไปนี้:

* ลิตร - 0.2642 (1/3.785)
* แกลลอนสหรัฐ - 1
* แกลลอนอิมพีเรียล - 1.20095
