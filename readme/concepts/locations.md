# สถานที่ตั้ง

แอปพลิเคชัน UMS ใช้ลำดับชั้นสถานที่ตั้งมาตรฐานของ iviva

ต้นไม้สถานที่ตั้งทั้งหมดที่คุณกำหนดในแอป iviva จะมีให้ใช้ใน UMS



คุณสามารถแนบมิเตอร์, มิเตอร์เสมือนและกลุ่มมิเตอร์ไปยังสถานที่ตั้ง คุณยังสามารถกำหนดคุณสมบัติเพิ่มเติมสำหรับแต่ละสถานที่ตั้ง

จากมุมมองของส่วนต่อประสานผู้ใช้, สถานที่ตั้งจะแสดงเป็นเอนทิตี้แยกต่างหาก แต่จะถูกจัดการภายในเป็นแท็ก อย่างไรก็ตาม พวกเขามีความพิเศษในทางที่คุณไม่จำเป็นต้องสร้างแท็กด้วยตนเอง พวกเขาจะถูกซิงค์จากแอปพลิเคชันสถานที่ตั้งของ iviva
