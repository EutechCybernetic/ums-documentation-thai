# สถานที่

แอปพลิเคชัน UMS ใช้ลำดับชั้นสถานที่มาตรฐานของ iviva

ต้นไม้สถานที่ทั้งหมดที่คุณกำหนดในแอป iviva สามารถใช้ได้ใน UMS



คุณสามารถแนบเมตร, เมตรเสมือน และกลุ่มเมตรไปยังสถานที่ คุณยังสามารถกำหนดคุณสมบัติเพิ่มเติมสำหรับแต่ละสถานที่

จากมุมมองของส่วนต่อประสานผู้ใช้, สถานที่แสดงเป็นเอ็นทิตี้แยกต่างหาก แต่จัดการภายในเป็นแท็ก อย่างไรก็ตาม พวกเขามีความพิเศษในทางที่คุณไม่ต้องสร้างแท็กด้วยตนเอง พวกเขาถูกซิงค์จากแอปพลิเคชันสถานที่ของ iviva
