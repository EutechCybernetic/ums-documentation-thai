# ภาพรวม

แอปพลิเคชัน UMS สามารถจัดการหลายประเภทของสาธารณูปโภค ดังที่กล่าวไว้ในส่วนก่อนหน้านี้

ข้อมูลการใช้งาน - รวมถึงการอ่านมิเตอร์ - สามารถป้อนเข้าสู่ระบบผ่านทาง API หรือ Mqtt การรวมตัวอื่น ๆ ยังสามารถใช้ได้สำหรับการรับข้อมูล โดยใช้ Lucy สามารถอ่านข้อมูลการใช้งานจากใบแจ้งหนี้ PDF และภาพ และป้อนเข้าสู่แอปพลิเคชันได้ด้วย

เมื่อข้อมูลถูกนำเข้าแล้ว แอปพลิเคชันจะคำนวณและจัดเก็บข้อมูลการใช้งานและเมตริกอื่น ๆ โดยใช้การแก้ไขเชิงเส้นที่จำเป็น

คุณจึงสามารถติดตามข้อมูลการใช้งานในหลายมิติ รวมถึงช่วงวันที่ แท็ก และเส้นฐาน
