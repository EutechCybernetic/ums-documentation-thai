# ภาพรวม

แอปพลิเคชัน UMS สามารถจัดการกับหลายประเภทของสาธารณูปโภค ดังที่กล่าวไว้ในส่วนก่อนหน้านี้

ข้อมูลการใช้งาน - รวมถึงการอ่านมิเตอร์ - สามารถป้อนเข้าสู่ระบบผ่านทาง API หรือ Mqtt รวมถึงการรวมตัวอื่น ๆ ที่สามารถใช้ในการรับข้อมูล โดยใช้ Lucy คุณสามารถอ่านข้อมูลการใช้งานจากใบแจ้งหนี้ PDF และภาพ และป้อนเข้าสู่แอปพลิเคชันด้วย

เมื่อข้อมูลถูกนำเข้าแล้ว แอปพลิเคชันจะคำนวณและจัดเก็บข้อมูลการใช้งานและเมตริกส์อื่น ๆ โดยใช้การแก้ไขที่จำเป็น

คุณจึงสามารถติดตามข้อมูลการใช้งานในหลายมิติ รวมถึงช่วงวันที่ แท็ก และเส้นฐาน
