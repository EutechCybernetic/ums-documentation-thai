# ภาพรวม

แอปพลิเคชัน UMS สามารถจัดการกับหลายประเภทของสาธารณูปโภค ดังที่กล่าวไว้ในส่วนก่อนหน้านี้

ข้อมูลการใช้งาน - รวมถึงการอ่านมิเตอร์ - สามารถป้อนเข้าสู่ระบบได้ผ่านทาง API หรือ Mqtt การรวมตัวอื่น ๆ ก็สามารถใช้สำหรับการรับข้อมูล โดยใช้ Lucy สามารถอ่านข้อมูลการใช้งานจากใบแจ้งหนี้ PDF และรูปภาพ และป้อนเข้าสู่แอปพลิเคชันได้ด้วย

เมื่อข้อมูลถูกนำเข้า แอปพลิเคชันจะคำนวณและจัดเก็บข้อมูลการใช้งานและเมตริกอื่น ๆ โดยใช้การแก้ไขเชิงเส้นที่จำเป็น

คุณจึงสามารถติดตามข้อมูลการใช้งานในหลายมิติ รวมถึงช่วงวันที่ แท็ก และเส้นฐาน
