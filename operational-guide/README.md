# คู่มือการทำงาน

งานที่สำคัญในการดำเนินการของแอปพลิเคชัน UMS คือการสามารถติดตามและรายงานข้อมูลการใช้งานของสาธารณูปโภคต่างๆ

แอปพลิเคชันนี้มีเครื่องมือ 3 อย่างสำหรับทำงานนี้:

1. ตัวสำรวจมิเตอร์
2. ตัวสำรวจแท็ก
3. แดชบอร์ด

## การเข้าถึงเครื่องมือเหล่านี้

เครื่องมือดำเนินการถูกจำแนกตามประเภทของมิเตอร์\
แต่ละประเภทมิเตอร์มีส่วนที่แยกต่างหากสำหรับสำรวจมิเตอร์และแท็ก

แต่แดชบอร์ดนั้นถูกรวมกันและสามารถปรับแต่งได้ และสามารถเข้าถึงได้จากเมนูแดชบอร์ด

เครื่องมือทั้งหมดนี้สามารถเข้าถึงได้จากแถบด้านข้าง
