# คู่มือการทำงาน

งานที่สำคัญในการทำงานของแอปพลิเคชัน UMS คือการสามารถติดตามและรายงานข้อมูลการใช้งานในหลากหลายสาขาอุตสาหกรรม

แอปพลิเคชันนี้มีเครื่องมือ 3 อย่างสำหรับการทำงานนี้:

1. ตัวสำรวจมิเตอร์
2. ตัวสำรวจแท็ก
3. แดชบอร์ด

## การเข้าถึงเครื่องมือเหล่านี้

เครื่องมือการทำงานถูกจัดหมวดหมู่ตามประเภทของมิเตอร์\
แต่ละประเภทมิเตอร์มีส่วนที่แยกจากกันเพื่อสำรวจมิเตอร์และแท็ก

แต่แดชบอร์ดนั้นถูกรวมกันและสามารถปรับแต่งได้ และสามารถเข้าถึงได้จากเมนูแดชบอร์ด
