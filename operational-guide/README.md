# คู่มือการดำเนินงาน

งานดำเนินการหลักในแอปพลิเคชัน UMS คือการสามารถติดตามและรายงานข้อมูลการใช้งานของสิ่งอำนวยความสะดวกต่างๆ

แอปพลิเคชันนี้มีเครื่องมือ 3 อย่างสำหรับการทำงานนี้:

1. ตัวสำรวจมิเตอร์
2. ตัวสำรวจแท็ก
3. แดชบอร์ด

## การเข้าถึงเครื่องมือเหล่านี้

เครื่องมือดำเนินการถูกจัดหมวดหมู่ตามประเภทมิเตอร์\
แต่ละประเภทมิเตอร์มีส่วนที่แยกออกเพื่อสำรวจมิเตอร์และแท็ก

แต่แดชบอร์ดนั้นถูกรวมและสามารถปรับแต่งได้ และสามารถเข้าถึงได้จากเมนูแดชบอร์ด

เครื่องมือทั้งหมดนี้สามารถเข้าถึงได้จากแถบด้านข้าง
