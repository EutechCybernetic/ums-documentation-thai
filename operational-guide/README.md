# คู่มือการดำเนินงาน

งานดำเนินการหลักในแอปพลิเคชัน UMS คือการสามารถติดตามและรายงานข้อมูลการใช้งานของสิ่งอำนวยความสะดวกต่างๆ

แอปพลิเคชันนี้มีเครื่องมือ 3 ชนิดสำหรับการทำงานนี้:

1. เครื่องมือสำรวจมิเตอร์
2. เครื่องมือสำรวจแท็ก
3. แดชบอร์ด

## การเข้าถึงเครื่องมือเหล่านี้

เครื่องมือดำเนินการถูกจัดหมวดหมู่ตามประเภทของมิเตอร์\
แต่ละประเภทมิเตอร์มีส่วนที่แยกต่างหากสำหรับสำรวจมิเตอร์และแท็ก

อย่างไรก็ตาม, แดชบอร์ดถูกรวมเข้าด้วยกันและสามารถปรับแต่งได้ และสามารถเข้าถึงได้จากเมนูแดชบอร์ด

เครื่องมือเหล่านี้ทั้งหมดสามารถเข้าถึงได้จากแถบข้าง
