# สำรวจแท็ก

สำรวจแท็กคือที่ที่คุณสามารถวิเคราะห์การบริโภคตามแท็กที่คุณได้กำหนดไว้

ลำดับชั้นของตำแหน่งที่ตั้ง (ที่กำหนดไว้ในแอปพลิเคชัน Location ของ iviva) จะแสดงที่นี่ และให้คุณวิเคราะห์ข้อมูลที่แต่ละตำแหน่งในลำดับชั้น

{% hint style="info" %}
ดู [marking-the-main-meter.md](../getting-started/configuring-the-application/marking-the-main-meter.md "mention") สำหรับข้อมูลเกี่ยวกับการรับข้อมูลการบริโภคตามตำแหน่งที่ถูกต้อง
{% endhint %}


สำรวจแท็กจะมีลักษณะดังนี้:

<figure><img src="../.gitbook/assets/image (15).png" alt=""><figcaption></figcaption></figure>

ทางด้านซ้ายคุณจะมีแท็กทั้งหมดของคุณ ซึ่งจัดระเบียบในโครงสร้างต้นไม้และแยกตามประเภทแท็ก

คุณสามารถคลิกที่แท็กใด ๆ เพื่อดูข้อมูลการบริโภคเกี่ยวกับแท็กนั้น ด้านขวาจะแสดง [consumption-analysis-tool.md](consumption-analysis-tool.md "mention") เพื่อดูข้อมูลการบริโภคเกี่ยวกับแท็กนั้น


คุณสามารถเลือกแท็กเดียวจากแต่ละประเภทแท็กและดูข้อมูลการบริโภคเกี่ยวกับมัน หากคุณเลือกแท็กแม่ ข้อมูลการบริโภคจะมีข้อมูลที่รวมมาจากแท็กลูกทั้งหมดด้วย

คุณยังสามารถรวมแท็กจากประเภทแท็กที่แตกต่างกัน


ในตัวอย่างด้านล่าง:

<figure><img src="../.gitbook/assets/image (16).png" alt=""><figcaption></figcaption></figure>

เราได้เลือกดูข้อมูลการบริโภคสำหรับรายการที่แท็ก 'อุปกรณ์' ทั้งหมดที่ยังมีตำแหน่งที่ตั้งเป็น 'Precinct1'

คุณสามารถเลือกแท็กเดียวจากแต่ละกลุ่มแท็กและดูข้อมูลที่ตัดกันทั้งหมดของแท็กที่เลือก

คุณสามารถยกเลิกการเลือกแท็กโดยคลิกที่แท็กอีกครั้ง


หากคุณเลื่อนลงด้านล่างเครื่องมือวิเคราะห์การบริโภค คุณสามารถดูรายการของมิเตอร์ มิเตอร์เสมือนและกลุ่มมิเตอร์ที่สนับสนุนแท็กที่เลือก (หรือแท็ก)
