# เครื่องมือวิเคราะห์การใช้งาน

เครื่องมือวิเคราะห์การใช้งานให้คุณมีส่วนต่อประสานเพื่อวิเคราะห์การใช้งานของวัตถุใด ๆ (เมตร, กลุ่ม หรือแท็ก) ในหลาย ๆ วิธี

{% hint style="info" %}
เมื่อเราพูดถึง 'วัตถุ' เราหมายถึงเมตร, เมตรเสมือนจริง, กลุ่มเมตร หรือแท็ก
{% endhint %}



เครื่องมือนี้สามารถใช้งานได้ในหลาย ๆ วิธี:

1. คลิกที่เมตรภายในตัวสำรวจเมตร
2. บนแผงด้านขวาในตัวสำรวจแท็กที่แสดงข้อมูลการใช้งานสำหรับแท็กที่เลือก
3. สามารถเพิ่มลงในแดชบอร์ดเป็นวิดเจ็ตและปรับแต่งได้

เครื่องมือมีลักษณะดังนี้:

<figure><img src="../.gitbook/assets/image (4).png" alt=""><figcaption></figcaption></figure>

1. แผงรายละเอียด - มีรายละเอียดเกี่ยวกับวัตถุเฉพาะ ซึ่งอาจจะมองเห็นหรือไม่มองเห็นขึ้นอยู่กับวิธีที่คุณเรียกใช้เครื่องมือ
2. แผงวิเคราะห์
3. ตัวกรองวันที่ - คุณสามารถเลือกช่วงวันที่เพื่อวิเคราะห์การใช้งาน
4. KPI ระดับสูง - ปัจจุบันแสดงการใช้งานที่สูงที่สุดและ (ถ้าเกี่ยวข้อง) ความเบี่ยงเบนทั้งหมดจากเส้นฐาน
   1. คุณสามารถส่งออกข้อมูลที่กำลังดูได้ทุกเวลาโดยใช้ปุ่มส่งออก มันจะดาวน์โหลดการส่งออก csv ของข้อมูล
5. แผงตัวกรอง - คลิกที่นี่เพื่อขยายเครื่องมือเพื่อแสดงตัวกรองที่มีทั้งหมด
6. [consumption-contribution-tool.md](consumption-contribution-tool.md "mention")- นี้เกี่ยวข้องเฉพาะกับเมตรเสมือนจริงและกลุ่มเมตร

## ช่วงวันที่

เมื่อเลือกช่วงวันที่, คุณสามารถเลือกช่วงวันที่สัมพันธ์หรือช่วงวันที่ที่กำหนดเอง

<figure><img src="../.gitbook/assets/Screenshot 2025-01-29 at 20.58.08.png" alt="" width="369"><figcaption></figcaption></figure>

เมื่อคุณเลือกช่วงวันที่สัมพันธ์, คุณสามารถเลือกที่จะแสดงข้อมูลสำหรับ

1. 7 วันที่ผ่านมา
2. 30 วันที่ผ่านมา
3. 365 วันที่ผ่านมา

ช่วงวันที่ที่เลือกจะเป็นอัตโนมัติตามที่คุณเลือก

คุณยังสามารถเลือก 'กำหนดเอง' ในกรณีนั้นคุณสามารถระบุวันที่เริ่มต้นและวันที่สิ้นสุด

คลิกที่วันที่เริ่มต้นในปฏิทินก่อน, แล้วคลิกวันที่สิ้นสุด- ปฏิทินควรจะเน้นวันที่ทั้งหมดระหว่างวันที่เริ่มต้นและวันที่สิ้นสุด

{% hint style="info" %}
การเปรียบเทียบระยะเวลาปัจจุบันกับระยะเวลาก่อนหน้านี้สามารถทำได้เฉพาะเมื่อคุณเลือกช่วงวันที่สัมพันธ์ - เพื่อให้เครื่องมือรู้วันที่สำหรับเธอด้วย
{% endhint %}

## ตัวกรองและการวิเคราะห์ที่ขยาย

คุณสามารถคลิกที่ไอคอน 'ตัวกรอง' ที่ด้านบนขวา (#6 ในแผนภาพด้านบน) เพื่อขยายเพื่อดูส่วนต่อประสานของการวิเคราะห์แบบเต็ม

<figure><img src="../.gitbook/assets/image (17).png" alt=""><figcaption></figcaption></figure>

นี่คล้ายกับส่วนต่อประสานก่อนหน้านี้ แต่มีตัวเลือกเพิ่มเติม



1. ความสามารถในการเปลี่ยนแปลงประเภทการแสดงผล คุณสามารถเลือกระหว่าง
   1. แผนภูมิแท่ง (มุมมองเริ่มต้น)
   2. แผนภูมิคอลัมน์ (แท่งวางแนวนอน)
   3. แผนภูมิพื้นที่
   4. แผนภูมิเส้น
   5. แผนภูมิโดนัท (แผนภูมิวงกลม - ไม่เกี่ยวข้องสำหรับบางประเภทของการวิเคราะห์)
2. แผงตัวกรองทางด้านขวาที่ให้คุณมีตัวเลือกตัวกรองเพิ่มเติม



## ตัวเลือกการกรอง

แผงตัวกรองให้คุณมีช่วงตัวกรองและการจัดกลุ่มและตัวเลือกการรวมข้อมูลสำหรับข้อมูล

ไม่ใช่ตัวกรองทั้งหมดที่เกี่ยวข้องเสมอ



### ประเภทยูทิลิตี้

เลือกประเภทยูทิลิตี้ที่คุณต้องการวิเคราะห์

&#x20;ตัวเลือกนี้สามารถใช้ได้เมื่อคุณกำหนดค่าเครื่องมือจากแดชบอร์ด&#x20;

เมื่อคุณเข้าถึงเครื่องมือจากเมตรหรือแท็ก, ประเภทยูทิลิตี้ที่รู้จักแล้วดังนั้นตัวเลือกนั้นจะไม่สามารถใช้งานได้ในกรณีเหล่านั้น



### หน่วย

ขึ้นอยู่กับประเภทยูทิลิตี้ - รายการของหน่วยที่เป็นไปได้จะถูกแสดงด้วยหน่วยฐานเริ่มต้นที่เลือก

คุณสามารถเปลี่ยนหน่วยที่ข้อมูลนำเสนอ



### ประเภทเป้าหมาย

{% hint style="info" %}
ตัวเลือกนี้สามารถใช้ได้เมื่อคุณเข้าถึงเครื่องมือวิเคราะห์การใช้งานจากวิดเจ็ตแดชบอร์ด
{% endhint %}

เลือกวัตถุประเภทใดที่คุณต้องการวิเคราะห์

นี้สามารถเป็นหนึ่งใน:

1. เมตรยูทิลิตี้
2. เมตรเสมือนจริง
3. กลุ่มเมตร
4. แท็ก

### เป้าหมาย

{% hint style="info" %}
ตัวเลือกนี้สามารถใช้ได้เมื่อคุณเข้าถึงเครื่องมือวิเคราะห์การใช้งานจากวิดเจ็ตแดชบอร์ดและหลังจากที่คุณเลือก 'ประเภทเป้าหมาย'
{% endhint %}

เมื่อคุณเลือกประเภทเป้าหมายของคุณ, ขึ้นอยู่กับการเลือกคุณมีตัวเลือกที่แตกต่างกัน

#### เมตรยูทิลิตี้

ถ้าคุณเลือก 'เมตรยูทิลิตี้' เป็นประเภทเป้าหมาย, คุณสามารถเลือกเมตรยูทิลิตี้ที่คุณต้องการวิเคราะห์ คุณสามารถเลือกเมตรยูทิลิตี้หลาย ๆ ตัวด้วย

#### เมตรเสมือนจริง

ถ้าคุณเลือก 'เมตรเสมือนจริง' เป็นประเภทเป้าหมาย, คุณสามารถเลือกเมตรเสมือนจริงที่คุณต้องการวิเคราะห์ คุณสามารถเลือกเมตรเสมือนจริงหลาย ๆ ตัวด้วย

#### กลุ่มเมตร

ถ้าคุณเลือก 'กลุ่มเมตร' เป็นประเภทเป้าหมาย, คุณสามารถเลือกกลุ่มที่คุณต้องการวิเคราะห์ คุณสามารถเลือกกลุ่มหลาย ๆ กลุ่มเพื่อวิเคราะห์ด้วย

#### แท็ก

ถ้าคุณเลือก 'แท็ก' เป็นประเภทเป้าหมาย, คุณสามารถเลือกแท็กเฉพาะที่คุณต้องการวิเคราะห์

คุณจะถูกขอให้เลือกประเภทแท็กก่อน, แล้วจึงนำเสนอต้นไม้แท็กเพื่อเลือกแท็กของคุณ

คุณสามารถเลือกแท็กหลาย ๆ แท็กและข้อมูลที่แสดงจะเป็นจาก _การตัดกัน_ ของทุก ๆ แท็ก

ตัวอย่างเช่น, ถ้าคุณเลือก \`Location:Building1\` และ \`System:PlugLoad\`

การใช้งานจะถูกแสดงสำหรับเมตรที่มีแท็กทั้งสอง

{% hint style="info" %}
แท็กย่อยจะถูกพิจารณาด้วยเนื่องจากการใช้งานของแต่ละแท็กจะรวมการใช้งานจากแท็กย่อยโดยอัตโนมัติ
{% endhint %}

ใช้ตัวอย่างด้านบน, เมตรที่มีเครื่องหมายถูกสีเขียวจะถูกเลือกโดยใช้การผสมของ Location:Building1 และ System:PlugLoad

### การเปรียบเทียบรายการหลาย ๆ รายการ

คุณสามารถเปรียบเทียบรายการหลาย ๆ รายการ (เรียกว่า 'เป้าหมาย').

เพื่อทำเช่นนั้น - เลือกเป้าหมายเดียวก่อน

เมื่อคุณเลือกเป้าหมาย, คุณจะมีตัวเลือกเพื่อเลือกเป้าหมายเพิ่มเติมเพื่อเปรียบเทียบ

<figure><img src="../.gitbook/assets/image (20).png" alt="" width="375"><figcaption></figcaption></figure>

เมื่อคุณใช้ตัวเลือกนั้น - คุณจะได้รับหน้าจอที่คุณสามารถเพิ่มเป้าหมายเพิ่มเติม

แต่ละเป้าหมายอาจเป็นเมตร, เมตรเสมือนจริง, กลุ่ม หรือแท็ก&#x20;

<figure><img src="../.gitbook/assets/Screenshot 2025-02-12 at 16.19.01.png" alt=""><figcaption></figcaption></figure>

บนแผงด้านซ้าย, รายการของเป้าหมายเพิ่มเติมจะถูกแสดง

เมื่อคุณคลิกที่เป้าหมาย, บนแผงด้านขวา, คุณสามารถเปลี่ยนเป้าหมายโดยระบุประเภทเป้าหมายใหม่แล้วเลือกรายการเป้าหมายจริง

คุณสามารถเพิ่มเป้าหมายใหม่โดยคลิกที่ปุ่ม '+ เพิ่มเป้าหมาย' ที่ด้านบน



เมื่อคุณเพิ่มเป้าหมาย, วิดเจ็ตจะรีเฟรชเพื่อแสดงเป้าหมายที่อัปเดตเป็นแท่งใหม่หรือคอลัมน์หรือเวจใหม่ในแผนภูมิ

### ถัง

โดยค่าเริ่มต้น, ข้อมูลถูกเลือกตามช่วงวันที่และตัวกรองอื่น ๆ และสะสมเป็นช่วงเวลา 1 วัน

ช่วงเวลานี้เรียกว่า _ถัง_ มีถังอื่น ๆ ที่มีให้:

15 นาที, 30 นาที, 1 ชั่วโมง, วัน, สัปดาห์, เดือน และปี

**15 นาที** ถังจะแสดงข้อมูลในช่วง 15 นาที หมายความว่าสำหรับทุก ๆ ชั่วโมง, มันจะแสดง 4 ค่า:

3:00, 3:15, 3:30, 3:45 (ถ้าเป็นช่วงเวลา 3 ทุ่ม)



**30 นาที** ถังจะแสดงข้อมูลในช่วง 30 นาที - 3:00 และ 3:30



**วัน** ถังจะแสดงจุดข้อมูลหนึ่งต่อวัน&#x20;



**สัปดาห์** ถังแสดงจุดข้อมูลหนึ่งสำหรับทุก ๆ สัปดาห์ โปรดทราบว่าสัปดาห์ไม่ได้หมายความว่า '7 วัน' แต่มันหมายความว่า 'หนึ่งสัปดาห์'

ถ้าคุณอยู่กลางสัปดาห์ (เช่น, วันอังคาร) - มันจะแสดงข้อมูลจากวันอาทิตย์ถึงวันอังคารสำหรับสัปดาห์นั้น\
สำหรับสัปดาห์ก่อนหน้านี้จะแสดงข้อมูลจากวันอาทิตย์ถึงวันเสาร์



**เดือน** ถังจะจัดกลุ่มข้อมูลเป็นจุดข้อมูลหนึ่งสำหรับทุก ๆ เดือนของปี



**ปี** ถังจับกลุ่มข้อมูลสำหรับปีทั้งหมด

{% hint style="info" %}
ถ้าคุณต้องการดูการใช้งานสะสมทั้งหมดเป็นตัวเลขเดียว, การเลือก 'ปี' เป็นวิธีที่ดีเพราะมันจะรวมข้อมูลทั้งหมดสำหรับปีทั้งหมด
{% endhint %}



เมื่อจุดข้อมูลถูกจับกลุ่ม, ข้อมูลทั้งหมดจะถูกรวมเข้าด้วยกันและแสดงในจุดข้อมูลนั้น
