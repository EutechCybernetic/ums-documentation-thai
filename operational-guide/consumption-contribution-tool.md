# เครื่องมือวิเคราะห์การบริโภคพลังงาน

เครื่องมือวิเคราะห์การบริโภคพลังงานช่วยให้คุณเห็นว่าการบริโภคพลังงานสำหรับมิเตอร์เสมือนหรือกลุ่มมิเตอร์สร้างขึ้นมาจากมิเตอร์ที่อยู่เบื้องหลังอย่างไร

คุณสามารถเข้าถึงมันได้โดยไปที่เครื่องมือวิเคราะห์การบริโภคพลังงานสำหรับมิเตอร์เสมือนหรือกลุ่มที่เฉพาะเจาะจงและคลิกที่ปุ่ม 'ดูการสนับสนุน' ที่ด้านล่าง

<figure><img src="../.gitbook/assets/image (1).png" alt=""><figcaption></figcaption></figure>

นี่จะเปิดหน้าตาของการสนับสนุน

<figure><img src="../.gitbook/assets/Screenshot 2025-01-30 at 17.30.19.png" alt=""><figcaption></figcaption></figure>

ทางด้านซ้าย แถบแสดงมิเตอร์เสมือนหรือกลุ่มที่กำลังถูกตรวจสอบในขณะนี้

ทางด้านซ้าย มิเตอร์ที่อยู่เบื้องหลังที่สร้างขึ้นมาจากการบริโภคพลังงานของมิเตอร์เสมือนจะถูกแสดง

เส้นที่เชื่อมมิเตอร์เสมือนกับมิเตอร์ที่อยู่เบื้องหลังแทนการไหลของพลังงาน

ความหนาของเส้นเป็นอัตราส่วนต่อการบริโภคพลังงานของมิเตอร์เสมือนที่มิเตอร์นั้นสนับสนุนในช่วงวันที่เลือก

เส้นสีแดงแทนการบริโภคลบ (เช่น สูตรลบพลังงานนั้นออกจากมิเตอร์เสมือน)

เส้นสีเขียวแทนการบริโภคบวก

{% hint style="info" %}
ช่วงวันที่ในหน้าตาของการวิเคราะห์การบริโภคพลังงานใช้เพื่อแสดงค่าพลังงานในแผนภูมิการสนับสนุน
{% endhint %}

คุณสามารถเลื่อนเมาส์ไปที่แต่ละแถบเพื่อดูค่าการบริโภคพลังงานที่แท้จริงที่สนับสนุนโดยมัน

ถ้าแถบใด ๆ ที่ด้านซ้ายเป็นมิเตอร์เสมือนหรือกลุ่มเอง คุณสามารถดับเบิลคลิกที่มันเพื่อขยายมันออกเพื่อแสดงมิเตอร์ที่อยู่เบื้องหลังของมันเช่นกัน
