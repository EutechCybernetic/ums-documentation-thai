# เครื่องมือวิเคราะห์การบริโภคพลังงาน

เครื่องมือวิเคราะห์การบริโภคพลังงานช่วยให้คุณเห็นว่าการบริโภคพลังงานสำหรับมิเตอร์เสมือนหรือกลุ่มมิเตอร์นั้นสร้างขึ้นมาจากมิเตอร์ที่อยู่เบื้องหลังอย่างไร

คุณสามารถเข้าถึงมันได้โดยไปที่เครื่องมือวิเคราะห์การบริโภคพลังงานสำหรับมิเตอร์เสมือนหรือกลุ่มที่เฉพาะเจาะจงและคลิกที่ปุ่ม 'ดูการสนับสนุน' ที่ด้านล่าง

<figure><img src="../.gitbook/assets/image (1).png" alt=""><figcaption></figcaption></figure>

สิ่งนี้จะเปิดหน้าต่างส่วนที่แสดงการสนับสนุน

<figure><img src="../.gitbook/assets/Screenshot 2025-01-30 at 17.30.19.png" alt=""><figcaption></figcaption></figure>

ทางด้านซ้าย แถบแสดงมิเตอร์เสมือนหรือกลุ่มที่กำลังถูกตรวจสอบอยู่ในปัจจุบัน

ทางด้านซ้าย มิเตอร์ที่อยู่เบื้องหลังที่สร้างขึ้นมาเป็นการบริโภคพลังงานของมิเตอร์เสมือนถูกแสดงอยู่

เส้นที่เชื่อมมิเตอร์เสมือนกับมิเตอร์ที่อยู่เบื้องหลังของมันแทนที่จะเป็นการไหลของพลังงาน

ความหนาของเส้นเป็นสัดส่วนกับว่ามิเตอร์นั้นมีส่วนร่วมกับการบริโภคพลังงานของมิเตอร์เสมือนเพียงใดสำหรับช่วงวันที่เลือก

เส้นสีแดงแทนการบริโภคลบ (เช่น สูตรลบพลังงานจากมิเตอร์เสมือน)

เส้นสีเขียวแทนการบริโภคบวก

{% hint style="info" %}
ช่วงวันที่ในหน้าต่างวิเคราะห์การบริโภคพลังงานใช้เพื่อแสดงค่าพลังงานในแผนภูมิการสนับสนุน
{% endhint %}

คุณสามารถวางเมาส์ทับแถบแต่ละแถบเพื่อดูค่าการบริโภคพลังงานจริงที่สนับสนุนโดยมัน

หากแถบใด ๆ ที่ด้านซ้ายเป็นมิเตอร์เสมือนหรือกลุ่มเอง คุณสามารถดับเบิลคลิกที่มันเพื่อขยายมันออกเพื่อแสดงมิเตอร์ที่อยู่เบื้องหลังของมันด้วย
