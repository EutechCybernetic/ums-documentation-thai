# เครื่องมือวิเคราะห์การบริโภค

เครื่องมือวิเคราะห์การบริโภคช่วยให้คุณเห็นวิธีการสร้างการบริโภคสำหรับมิเตอร์เสมือนหรือกลุ่มมิเตอร์จากมิเตอร์ที่อยู่ภายใต้

คุณสามารถเข้าถึงมันได้โดยไปที่เครื่องมือวิเคราะห์การบริโภคสำหรับมิเตอร์เสมือนหรือกลุ่มที่เฉพาะเจาะจงและคลิกที่ปุ่ม 'ดูการสนับสนุน' ที่ด้านล่าง

<figure><img src="../.gitbook/assets/image (1).png" alt=""><figcaption></figcaption></figure>

นี่จะเปิดหน้าต่างส่วนต่อประสานการสนับสนุน

<figure><img src="../.gitbook/assets/Screenshot 2025-01-30 at 17.30.19.png" alt=""><figcaption></figcaption></figure>

ทางด้านซ้าย แถบแสดงมิเตอร์เสมือนหรือกลุ่มที่กำลังถูกตรวจสอบในขณะนี้

ทางด้านซ้าย มิเตอร์ที่อยู่ภายใต้ที่สร้างขึ้นมาจากการบริโภคของมิเตอร์เสมือนจะถูกแสดง

เส้นที่เชื่อมมิเตอร์เสมือนกับมิเตอร์ที่อยู่ภายใต้แสดงการไหลของพลังงาน

ความหนาของเส้นเป็นสัดส่วนกับว่ามิเตอร์นั้นมีส่วนร่วมเท่าไหร่ในการบริโภคพลังงานของมิเตอร์เสมือนสำหรับช่วงวันที่เลือก

เส้นสีแดงแสดงการบริโภคลบ (เช่น สูตรลบพลังงานนั้นออกจากมิเตอร์เสมือน)

เส้นสีเขียวแสดงการบริโภคบวก

{% hint style="info" %}
ช่วงวันที่ในหน้าต่างวิเคราะห์การบริโภคจะถูกใช้เพื่อแสดงค่าพลังงานในแผนภูมิการสนับสนุน
{% endhint %}

คุณสามารถวางเมาส์บนแต่ละแถบเพื่อดูค่าการบริโภคพลังงานที่จริงที่สนับสนุนโดยมัน

ถ้าแถบใดแถบหนึ่งทางด้านซ้ายเป็นมิเตอร์เสมือนหรือกลุ่มเอง คุณสามารถดับเบิลคลิกที่มันเพื่อขยายมันออกแสดงมิเตอร์ที่อยู่ภายใต้ของมันเช่นกัน
